Geometria Algébrica

Aula 4 - Curvas Projetivas

O plano projetivo (ou espaço projetivo bidimensional) sobre um corpo \(K\), denotado por \(\bbP^2\) é definido como sendo o conjunto de todas as retas passando por \((0,0,0) \in \bbA^3\). Observe que cada ponto em \(\bbA^3\setminus\{(0,0,0)\}\) determina uma única reta. Assim, denotamos por \([x_1:x_2:x_3]\) a reta determinada por \((x_1,x_2,x_3) \in \bbA^3\). Com isso, temos que \(\bbP^2 = \{[x_1:x_2:x_3] \mid \) algum \(x_i \neq 0\}\). Podemos identificar \(\bbP^2\) com conjunto de classes de equivalência \(\rfrac{\bbA^3\setminus\{(0,0,0)\}}{\overset{R}{\sim}}\), onde

\((x_1, x_2, x_3) \overset{R}{\sim} (y_1, y_2, y_3) \quad\Leftrightarrow\quad \text{ existe } \lambda \in K \text{ tal que } x_i = \lambda y_i\).

Podemos identificar o plano afim contido no plano projetivo da seguinte maneira

\(\bbA^2 = \{(x_1, x_2)\} \equiv \{[x_1:x_2:1]\} \subseteq \{[x_1:x_2:x_3]\} = \bbP^2\)

Um ponto em \(\bbP^2\setminus\bbA^2\) é chamado de ponto no infinito. A denominação se justifica. Vejamos. Seja \([0:1:1] \in \bbA^2 \subseteq \bbP^2\). Defina \(p_n = (0, n, 1) \in \bbA^3\). Note que esses pontos estão sobre o plano \(x_3 = 1\) e o plano \(x_1 = 0\). Veja, agora, que \([p_n] = [0:n:1] = [0:1:\frac{1}{n}]\) que tende a \([p_{\infty}] = [0:1:0]\) que é o eixo \(x_2\).

Observe que cada reta no plano \(x_3 = 1\) é a interseção desse plano com outro passando pela origem. Tal plano passando pela origem admite uma equação da forma \(a_1x_1 + a_2x_2 + a_3x_3 = 0\), onde \(a_1 \neq 0\) ou \(a_2 \neq 0\). Isso motiva a seguinte definição: uma reta no plano projetiva é um subconjunto de \(\bbP^2\) da forma \(\{[x_1:x_2:x_3] \mid a_1x_1 + a_2x_2 + a_3x_3 = 0\), onde os \(a_i\) são constantes não todas nulas\(\}\). A reta \(x_3 = 0\) é a reta no infinito. Cada reta distinta da reta no infinito a corta em exatamente um ponto.

Dizemos que duas formas não constante \(F,G \in K[X, Y, Z]\) são equivalentes, se existe \(\lambda \in K\) não nulo tal que \(G = \lambda F\). Uma curva plana projetiva é uma classe de equivalência de uma forma. O grau da curva é o grau que define a forma.

Seja \(R\) um domínio. Se \(F \in R[X_1,\dots,X_{n+1}]\) é uma forma, definimos a desomogeneização de \(F\) com respeito a \(X_{n+1}\) o polinômio \(F_{\ast} = F(X_1,\dots,X_n,1)\). Inversamente, se \(F \in R[X_1,\dots,X_n]\) é um polinômio de grau \(d\), escrito na forma \(F = F_0 + F_1 + \cdots + F_d\), onde \(F_i\) é uma forma de grau \(i\), definimos a homogeneização de de \(F\) com respeito a \(X_{n+1}\) o polinômio \(F^{\ast} = X_{n+1}^{d}F_0 + X_{n+1}^{d-1}F_1 + \cdots + F_d = \) \(X_{n+1}^{d}F(\frac{X_1}{X_{n+1}}, \dots, \frac{X_n}{X_{n+1}})\).

O fecho projetivo de uma curva definida por uma equação polinomial de grau \(d\), \(f(x_1,x_2) := \sum_{i+j\le d}a_{ij}x_1^ix_2^j\) é obtido pela homogeneização de \(f\) com respeito a \(x_3\), isto é, é o polinômio \(\sum_{i+j\le d}a_{ij}x_1^ix_2^jx_3^{d-i-j}\)

Em geral, uma curva de grau \(d\) admite \(d\) pontos no infinito, contados com multiplicidades.

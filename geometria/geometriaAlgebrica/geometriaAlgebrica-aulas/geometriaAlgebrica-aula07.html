Geometria Algébrica

Aula 7 - Morfismo entre variedades e dimensão

A topologia de Zariski em \(X = \bbP^{n_1} \times \cdots \times \bbP^{n_r} \times \bbA^{m}\) é a topologia onde \(U \subseteq X\) é aberto, se \(X\setminus U\) é um subconjunto algébrico de \(X\).

Seja \(V\) um conjunto algébrico irredutível não vazio de \(\bbP^{n_1} \times \cdots \times \bbP^{n_r} \times \bbA^{m}\). Qualquer subconjunto aberto \(U\) de \(V\) é chamado de variedade.

Sejam \(X\) uma variedade e \(U\) um subconjunto aberto não vazio de \(X\). Definimos \(\Gamma(U, \calO_X)\) para ser o conjunto das funções racionais sobre \(X\) que são definidas em cada \(P \in U\), isto é, \(\Gamma(U, \calO_X) = \cap_{P \in U}\calO_P(X)\).

Sejam \(X, Y\) variedades. Um morfismo de \(X\) para \(Y\) é um mapa \(\varphi: X \to Y\) tal que

  1. \(\varphi\) é contínuo;
  2. para cada aberto \(U\) de \(Y\), se \(f \in \Gamma(U, \calO_Y)\), então \(f\circ \varphi \in \Gamma(\varphi^{-1}(U), \calO_X)\).

Se \(\varphi: X \to Y\) é um morfismo entre variedades, então \(V = \cup V_{\alpha}\) e \(W = \cup W_{\alpha}\), onde \(V_{\alpha}\) e \(W_{\alpha}\) são abertos tais que

  1. \(\varphi(V_{\alpha}) \subseteq W_{\alpha}\);
  2. cada \(V_{\alpha}\) e \(W_{\alpha}\) são variedades afim;
  3. cada restrição \(\varphi|_{V_{\alpha}}: V_{\alpha} \to W_{\alpha}\) é um polinômio.

  1. Toda variedade \(V\) admite uma base de abertos isomorfos a variedades afins.
  2. Uma subvariedade fechada de uma variedade afim é também uma variedade afim.
  3. Se \(V\) é afim, então existe uma base de abertos \((V_f)\), com \(f \in \Gamma(V)\), tais que

    4. \(\Gamma(V_f) \cong \rfrac{\Gamma(V)[T]}{\langle Tf - 1 \rangle} \cong \Gamma(V)[\frac{1}{f}]\)

Sejam \(V \subseteq A\), \(W \subseteq B\) subvariedades fechadas. Então \(V \times W\) é subvariedade fechada de \(A \times B\).

Seja \(F \subseteq K\) uma extensão de corpos. Então existe um subcorpo \(S \subseteq K\) tal que

  1. os elementos de \(S\) são algebricamente independentes sobre \(F\);
  2. a extensão \(K \subseteq K(S)\) (subcorpo gerado) é algébrica.
Diz-se então que \(S\) é uma base de transcendência de \(K\). Duas tais bases quaisquer têm a mesma cardinalidade, chamada de grau de transcendência, \(\trdeg_F K\).

Seja \(V\) uma variedade, \(K(V)\) seu corpo de funções racionais. A dimensão de \(V\) é definida como o grau de transcendência \(\trdeg_K K(V)\).

Se \(U\) é aberto não vazio, temos \(K(U) = K(V)\). De fato, se \(P \in U\) temos a igualdade de anéis locais \(\calO_P(V) = \calO_P(U)\) e mesmo corpo de frações. Logo, \(\dim U = \dim V\). Em particular, \(\dim \bbA^n = \dim \bbP^n = \cdots\).

Seja \(V\) uma variedade afim e \(f \in \Gamma(V)\). Seja \(W\) uma componente irredutível de \(V(f)\). Então \(\dim W = \dim V - 1\).

Sejam \(K\) o corpo de funções em \(1\) variável e \(x \in K\setminus F\) qualquer função não constante. Então

  1. \(K\) tem que ser algébrico sobre \(F(x)\) (qualquer função não constante, serve para dar uma base de transcendência);
  2. Se \(\char(F) = 0\), então existe um elemento \(y \in K\) tal que \(K = F(x, y)\), corpo de funções de uma curva plana;
  3. Se \(R \supseteq F\) é um domínio com corpo de frações \(K\) e \(\frakp \neq \zero\) é um ideal primo de R, então o homomorfismo natural \(F \subseteq R \twoheadrightarrow \rfrac{R}{\frakp}\) é um isomorfismo.

Álgebra Não Comutativa

Aula 22 - Teorema de Wedderburn-Malcev (parte 2)

Sejam \(A\) uma \(F\)-álgebra e \(B\) um \(A\)-bimódulo. Uma aplicação \(f: A \to B\) é uma derivação generalizada se \[ f(ab) = f(a)b + af(b) \] para todo \(a, b \in A\).

Considere \(C^{\infty}(\bbR)\) a \(\bbR\)-álgebra das funções de classe \(C^{\infty}\) e seja \[ \begin{array}{rl} \frac{d}{dx}: C^{\infty}(\bbR) &\to C^{\infty}(\bbR) \\ f &\mapsto \frac{df}{dx} \end{array} \] Então \(\frac{d}{dx}\) é \(\bbR\)-linear e \(\frac{d}{dx}(fg) = \frac{df}{dx}g + f\frac{dg}{dx}\), ou seja, \(\frac{d}{dx}\) é uma derivação generalizada.

Seja \(A\) uma \(F\)-álgebra de dimensão finita. Se \(A\) é separável, então, para alguma base \(\{a_1, \dots, a_n\}\), existem elementos \(a_1', \dots, a_n' \in A\) tais que

  1. \(\sum_{i=1}^{n}a_i'a_i = 1\);
  2. para todo \(a \in A\), \(a_ia = \sum_{j=1}^{n}\lambda_{ij}(a)a_{j}\), \(\lambda_{ij}(a) \in F\), implica que \(aa_i' = \sum_{j=1}^n\lambda_{ji}(a)a_j'\), \(i = 1, \dots, n\).

Seja uma \(F\)-álgebra de dimensão finita separável. Então todo fator de \(A\) cinde e toda derivação generalizada é interna.

Seja \(A\) uma \(F\)-álgebra de dimensão finita tal que \(\rfrac{A}{J(A)}\) é separável. Então existe uma subálgebra semissimples \(B\) de \(A\) tal que \(A = B \dot+ J(A)\). Além disso, se \(A\) é unitária e \(B_1\) e \(B_2\) são subálgebras de \(A\) tais que \(A = B_1 \dot+ J(A) = B_2 \dot+ J(A)\), então existe \(j \in J(A)\) tal que \(B_1 = (1-j)B_2(1-j)^{-1}\).

  1. Mostre que \(f: A \to B\) definida por \(f_b(a) = ab - ba := [a,b]\), para todo \(a \in A\), com \(b\) fixo, é uma derivação generalizada, chamada de derivação generalizada interna. (Solução)
  2. O conjunto \(\Der_{F}(A, B) = \{f: A \to B \mid \) é uma derivação generalizada\(\}\) tem estrutura de \(F\)-espaço vetorial. Denote \(\Der_{F}(A, A) = \Der_{F}(A)\).
    1. Mostre que, em \(\Der_{F}(A)\), a composta de duas derivações, em geral, não é uma derivação.
    2. Mostre que se \(f, g \in \Der_{F}(A)\), então \([f,g] := f\circ g - g\circ f \in \Der_{F}(A)\). (Solução)
  3. Mostre que o teorema de Wedderburn-Malcev é equivalente ao seguinte afirmação: "seja \(A\) uma \(F\)-álgebra de dimensão finita e \(B'\) uma subálgebra separável de \(\rfrac{A}{J(A)}\). Então \(A\) contém uma subálgebra \(B\) isomorfa a \(B'\)". (Solução)
  4. Seja \(A = B \dot+ J(A)\) uma decomposição de Wedderburn-Malcev de uma álgebra \(A\). Mostre que \(B\) é uma subálgebra semissimples maximal de \(A\), ou seja, \(B\) não está contida propriamente em nenhuma subálgebra semissimples de \(A\). (Solução)

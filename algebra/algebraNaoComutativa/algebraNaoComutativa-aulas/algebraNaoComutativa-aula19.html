Álgebra Não Comutativa

Aula 19 - O Teorema do Centralizador Duplo

Se \(D\) é uma álgebra de divisão de dimensão finita sobre o seu centro, então \(\dim_{\calZ(D)}(D)\) é um quadrado.

Se \(A\) é uma álgebra simples de dimensão finita sobre \(\calZ(A)\), então \(\dim_{\calZ(A)}(A)\) é um quadrado.

Se \(A\) é uma álgebra e \(B \subseteq A\), então o centralizador de \(B\) em \(A\) é o conjunto \(C_A(B) = C(B) = \{a \in A \mid ab = ba, \forall b \in B\}\).

É fácil verificar que \(C(B)\) é uma subálgebra de \(A\), para todo \(B \subseteq A\).

  1. Para todo \(B \subseteq A\), temos que \(B \subseteq C(C(B))\).
  2. Se \(A\) é uma \(F\)-álgebra central simples, então \(C(C(A)) = C(F) = A\).
  3. Seja \(M\) um \(R\)-módulo à direita. Se \(E = \End(M)\) (como grupo abeliano), então \(M\) possui uma estrutura de \(E\)-módulo. Temos que \(f: R \to \End(M)\), \(r \mapsto f_r(m) = mr\) é um homomorfismo, \(\im(f) = R^M\), \(\ker(f) = \Ann(M)\) e \(R^M \cong \rfrac{R}{\Ann(M)}\). Note que \(C(R^M) = \End_R(M)\). Se \(\alpha \in \End_R(M)\) e \(a \in R^M\), então \((\alpha a)(m) = \alpha(a(m)) = \alpha(mr) = \alpha(m)r = (a\alpha)(m)\). Se \(\alpha a = a \alpha\), então \(\alpha(a(m)) = (a \alpha)(m)\), logo \(\alpha(mr) = \alpha(m)r\). Assim, \(C(C(R^M)) = C(\End_R(M)) = R^M\).

Sejam \(A\) uma \(F\)-álgebra central simples de dimensão finita e \(B\) uma subálgebra simples de \(A\). Então

  1. \(C(B)\) é simples;
  2. \(\dim_F(A) = \dim_F(B)\dim_F(C(B))\);
  3. \(C(C(B)) = B\).

  1. Sejam \(A\) uma \(F\)-álgebra com unidade e \(B\) uma subálgebra de \(A\). Seja \(R = A \otimes_F B^{op}\). Mostre que
    1. a ação \(x(a \otimes b) = b \times a\), para todo \(a, x \in A\) e \(b \in B\), dá uma estrutura de \(R\)-módulo à direita a \(A\);
    2. \(C(B) \cong \End_R(A)\), com a ação definida no item anterior. (Solução)
  2. Se \(B\) é uma subálgebra central simples de uma álgebra central simples com unidade \(A\) de dimensão finita, então \(A \cong B \otimes_F C(B)\). (Solução)

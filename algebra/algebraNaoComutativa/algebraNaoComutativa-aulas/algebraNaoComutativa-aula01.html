Álgebra Não Comutativa

Aula 1 - Revisão de teoria de anéis (parte 1)

Um anel \(R\) é um conjunto não vazio munido de duas operações \[ \begin{array}{ccc} \begin{array}{rl} +: R \times R &\to R \\ (a,b) &\mapsto a + b \end{array} & \begin{array}{rl} \cdot: R \times R &\to R \\ (a,b) &\mapsto a \cdot b := ab \end{array} \end{array} \] onde \(+\) é dita soma ou adição e \(\cdot\) é dita multiplicação ou produto, tais que, para todo \(a, b, c \in R\), valem as seguintes condições:

  1. comutatividade da soma: \(a + b = b + a\);
  2. associatividade da soma: \((a + b) + c = a + (b + c)\);
  3. existência do elemento neutro da soma: existe \(0 \in R\) tal que \(a + 0 = a\);
  4. existência de inversos aditivos: para todo \(a \in R\) existe \(-a \in R\) tal que \(a + (-a) = 0\);
  5. associatividade do produto: \((ab)c = a(bc)\);
  6. distributividade do produto com relação a soma à esquerda: \(a(b + c) = ab + ac\);
  7. distributividade do produto com relação a soma à direita: \((a + b)c = ac + bc\).
Dizemos que um anel \(R\) é comutativo, se o produto é comutativo, isto é, \(ab = ba\), para todo \(a, b \in R\). Dizemos que \(R\) é unitário (ou \(R\) é um anel com unidade), se existe um elemento neutro do produto diferente de \(0\), isto é, existe \(1 \in R\), com \(1 \neq 0\), tal que \(1a = a1 = a\), para todo \(a \in R\). Tal elemento é chamado de unidade.

  1. \(R\) denotará sempre, a menos menção contrária, um anel.
  2. Na definição de anel são únicos:
    1. o elemento neutro da soma, \(0\);
    2. os inversos aditivos, \(-a\), para cada \(a \in R\);
    3. o elemento neutro do produto, caso o anel seja unitário.
  3. Regras de sinais:
    1. \((-a)b = -ab = a(-b)\);
    2. \(-(-a) = a\);
    3. \((-a)(-b) = ab\).

  1. \(\bbZ, \bbQ, \bbR, \bbC\) são anéis comutativo com unidade.
  2. \(\bbZ_m = \{\overline{0}, \overline{1}, \dots, \overline{m - 1}\}\) é também anel comutativo com unidade.
  3. Denote \(M_n(R)\) o conjunto das matrizes \(n \times n\) com entradas em \(R\). Tal conjunto, munido das operações usuais de adição e multiplicação, é um anel que, em geral, não é comutativo. Se \(R\) tem unidade, então \(M_n(R)\) tem também unidade \[ I_n = (a_{ij}) = \begin{cases} 1, & \text{se } i = j, \\ 0, & \text{caso contrário,} \end{cases} \] chamada matriz identidade. Se \(R\) é unitário, denotamos por \(e_{ij} \in M_n(R)\) a matriz que possui \(1\) na entrada \((i,j)\) e \(0\) nas demais. Tais matrizes são chamandas de matrizes elementares. Assim, por exemplo, em \(M_3(R)\), temos que \[ e_{23} = \begin{pmatrix} 0 & 0 & 0 \\ 0 & 0 & 1 \\ 0 & 0 & 0 \end{pmatrix}. \] Usando as matrizes elementares podemos escrever \(I_n = \sum_{i=1}^n e_{ii}\). Além disso, observe também que \(e_{ij}e_{kl} = \delta_{jk}e_{i\ell}\), onde \( \delta_{jk} = \begin{cases} 1, & \text{se } j = k \\ 0, & \text{caso contrário.} \end{cases} \)
  4. \(UT_n(R) \subseteq M_n(R)\) conjunto das matrizes triangulares superiores \(n \times n\) (onde os elementos abaixo da diagonal principal são iguais a zero) é um anel com adição e multiplicação usuais.

Dizemos que \(R\) é um domínio, se \(ab = 0\), então \(a = 0\) ou \(b = 0\), para todo \(a, b \in R\). Nesse caso, dizemos também que \(R\) não possui divisores de zero. Se \(R\) é um domínio comutativo, então dizemos que \(R\) é um domínio de integridade.

  1. \(\bbZ, \bbQ, \bbR, \bbC\) são domínios de integridade.
  2. \(M_n(R)\) possui divisores de zero. Por exemplo, \(e_{ij}e_{kl} = 0\), se \(j \neq k\).

Seja \(R\) um anel com unidade e \(a \in R\).

  1. Dizemos que \(a\) possui inverso à direita (à esquerda), se existe \(b \in R\) tal que \(ab = 1\) (\(ba = 1\)).
  2. Dizemos que \(a\) é invertível, se existe \(b \in R\) tal que \(ab = ba = 1\).

Se \(a \in R\) é invertível, então seu inverso é único.

Seja \(V\) um espaço vetorial de dimensão infinita com base enumerável \(\{e_i\}\). Considere o conjunto \(\End(V) = \{f: V \to V \mid f \text{ é linear}\}\). Definindo as operações \((f + g)(v) = f(v) + g(v)\) e \((fg)(v) = f(g(v))\), temos que \(\End(V)\) é um anel que, em geral, não é comutativo com unidade. Considere os seguintes elementos \(f: V \to V\), \(e_i \mapsto e_{i+1}\), para todo \(i \ge 1\) e \(g: V \to V\), \(e_1 \mapsto 0\), \(e_i \mapsto e_{i-1}\), para todo \(i \ge 2\). Observe que \((gf)(e_i) = g(f(e_i)) = g(e_{i+1}) = e_i\), ou seja, \(g\) é uma inversa a esquerda de \(f\). Contudo, \(g\) não é inverso a direita \(f\), pois \((fg)(e_1) = f(g(e_1)) = f(0) = 0 \neq e_1\).

Se \(a \in R\) tem inversa à esquerda e à direita, então eles são iguais, isto é, se \(ab = 1\) e \(b'a = 1\), então \(b = b'\).

  1. Se \(R\) é uma anel com unidade, então o conjunto \(U(R) := \{a \in R \mid a \text{ é invertível}\}\) é um grupo, chamado de grupo das unidades de \(R\).
  2. Se \(U(R) = R \setminus\zero\), então dizemos que \(R\) é um anel de divisão. Um anel de divisão comutativo é chamado de corpo.

  1. \(\bbQ, \bbR, \bbC\) são corpos.
  2. \(\bbZ\) não é corpo, pois \(U(\bbZ) = \{-1, 1\}\).
  3. \(\bbZ_p\), com \(p\) primo, é um corpo. Na verdade, \(\bbZ_m\) é um corpo, se e somente se, \(m\) é primo.
  4. Sejam \(i, j, k\) símbolos. Defina \(\bbH := \{x_1 + x_2i + x_3j + x_4k \mid x_i \in \bbR\}\). Defina a adição em \(\bbH\), fazendo \[ (x_1 + x_2i + x_3j + x_4k) + (y_1 + y_2i + y_3j + y_4k) = x_1 + y_1 + (x_2 + y_2)i + (x_3 + y_3)j + (x_4 + y_4)k \] e defina a multiplicação em \(\bbH\) estendendo distributivamente a partir das seguintes regras \[ \begin{array}{cccc} i^2 = j^2 = k^2 = -1, & ij = k = -ji, & jk = i = -kj, & ki = j = -ik. \end{array} \] Assim, por exemplo, temos que \((1 + i + j)(2 - j - k) = 3 + i + 2j - 2k\). Com essas operações \(\bbH\) é um anel não comutativo com unidade. Seja \(\alpha = x_1 + x_2i + x_3j + x_4k\). Definimos \(\overline{\alpha} = x_1 - x_2i - x_3j - x_4k\), dito conjugado de \(\alpha\). Note que \(\alpha\overline{\alpha} = x_1^2 + x_2^2 + x_3^2 + x_4^2\). Chamamos tal produto de norma de \(\alpha\) e o denotamos por \(\norma{\alpha}\). Observe que \(\norma{\alpha} \ge 0\) e \(\norma{\alpha} = 0\), se e somente se, \(\alpha = 0\). Para todo \(\alpha \in \bbH \setminus \zero\), seja \(\alpha' = \frac{\overline{\alpha}}{\norma{\alpha}}\). Note que \(\alpha \alpha' = \alpha\frac{\overline{\alpha}}{\norma{\alpha}} = 1\). Assim, \(\bbH\) é um anel de divisão, chamado anel dos quatérnios reais.
  5. O conjunto \(R[x]\) dos polinômios com coeficientes em \(R\) com operações usuais de adição e multiplicação é um anel, chamado anel dos polinômios com coeficientes em \(R\). Observe que
    1. \(R[x]\) tem unidade, se \(R\) tem unidade;
    2. \(R[x]\) é comutativo, se \(R\) é comutativo;
    3. \(R[x]\) é um domínio, se \(R\) é um domínio,
  6. Seja \(C^0([a,b]) := \{f: [a, b] \to R \mid f \text{ é contínua}\}\). Dados \(f, g \in C^0([a,b])\), definindo as operações \((f + g)(x) = f(x) + g(x)\) e \((fg)(x) = f(x)g(x)\), para todo \(x \in [a, b]\), temos que \(C^0([a,b])\) é um anel comutativo e com unidade.

Sejam \(R\) um anel, \(a \in R\) e \(n \in \bbZ\). Definimos

  1. se \(n > 0\), \(na := \underbrace{a + \cdots + a}_{n \text{ vezes}}\);
  2. se \(n < 0\), \(na := |n|(-a)\);
  3. se \(n = 0\), \(0a = 0\).
Se existe um menor inteiro positivo \(n\) tal que \(na = 0\), para todo \(a \in R\), então dizemos que \(R\) é um anel de característica \(n\), o qual denotamos por \(\char(R) = n\). Caso tal \(n\) não exista, dizemos que \(R\) é um anel de característica zero.

  1. \(\char(\bbZ_m) = m\) e \(\char(\bbZ) = 0\).
  2. Se \(R\) é um anel com unidade de característica \(n\), então \(n\) é o menor inteiro positivo tal que \(n1 = 0\). Além disso, se \(R\) é um domínio com unidade, então \(\char(R)\) é primo. Em particular, a característica de um corpo ou é zero ou é um número primo.

Sejam \(R\) um anel e \(a \in R\). Dizemos que \(a\) é nilpotente, se existe \(n > 0\) tal que \(a^n := a \cdots a = 0\). Um anel onde todo elemento é nilpotente é chamado de anel nil.

  1. Se \(n \ge 2\), então \(M_n(R)\) possui elementos nilpotentes. Por exemplo, \(e_{ij}\), com \(i \neq j\).
  2. Matrizes triangulares estritamente superiores, \(ST_n(R)\), é um anel nil.

Dizemos que \(a \in R\) é idempotente se \(a^2 = a\).

Em \(M_n(R)\), temos que \(e_{ii} + e_{ij}\) é idempotente.

  1. Mostre que, em um anel com unidade, elementos invertíveis não são divisores de zero. Conclua que todo anel de divisão é um domínio.
  2. Mostre que todo domínio de integridade finito é um corpo.
    1. Mostre que se \(R\) é um anel comutativo e \(a, b \in R\) são nilpotentes, então \(a + b\) é nilpotente. Dê um exemplo mostrando que o resultado pode ser falso se \(R\) não é comutativo.
    2. Mostre que se \(a \in R\) é nilpotente, então \(1 \pm a\) é invertível. Determine o inverso dos elementos \(1 \pm a\).
  4. Mostre que se \(R\) possui somente elementos idempotentes (um anel dessa forma é chamado de anel booleano), então \(R\) é comutativo.
  5. Seja R um anel com unidade. Mostre que:
    1. idempotentes em \(R\) são divisores de zero.
    2. se \(R\) não possui elementos nilpotentes não triviais (ou seja, não nulos) todo idempotente em \(R\) é central, isto é, se \(e \in R\) e \(e^2 = e\), então \(er - re = 0\) para todo \(r \in R\).

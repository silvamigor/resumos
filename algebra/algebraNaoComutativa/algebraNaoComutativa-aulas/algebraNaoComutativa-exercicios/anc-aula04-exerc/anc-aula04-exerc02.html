Álgebra Não Comutativa

Aula 4 - Revisão de teoria de módulos (parte 1)

Sejam \(I \unlhd R\) (bilateral) e \(M\) um \(R\)-módulo. Mostre que \(\rfrac{M}{IM}\) tem estrutura de \(\rfrac{R}{I}\)-módulo com o produto dado por \((r + I)(m + IM) = rm + IM\).

Por \(I\) ser um ideal bilateral, temos que \(\rfrac{R}{I}\) é um anel. Além disso, sabemos que \(IM = \{\sum_{i=1}^{k}a_im_i \mid a_i \in I, m_i \in M, k \in \bbN\}\) é um \(R\)-submódulo de \(M\), logo \(\rfrac{M}{IM}\) é um \(R\)-módulo, conforme visto em aula. Vamos provar que ele é também um \(\rfrac{R}{I}\)-módulo, com o produto dado por \((r + I)(m + IM) = rm + IM\). Antes de mostrarmos que esse produto satisfaz a definição de módulo sobre um anel, vamos mostrar que ele está bem definido, quer dizer, que não depende do representante da classe. Assim, sejam \(r, r' \in R\) e \(m, m' \in M\) tais que \(r + I = r' + I\) e \(m + IM = m' + IM\). Queremos mostrar que \[ (r + I)(m + IM) := rm + IM = r'm' + IM =: (r' + I)(m' + IM), \] ou seja, queremos mostrar que \(rm - r'm' \in IM\). Observe que por \(r + I = r' + I\), temos que \(r - r' \in I\). Analogamente, por \(m + IM = m' + IM\), temos que \(m - m' \in IM\). Disso, segue que \((r - r')m \in IM\) e \(r'(m - m') \in IM\). Somando esses elementos, obtemos que \(rm - r'm' \in IM\), como queríamos. Feito isso, veja que, para todo \(r, r_1, r_2 \in R\) e para todo \(m, m_1, m_2 \in M\), temos que

  • \(\big((r_1 + I)(r_2 + I)\big)(m + IM) = \) \(\big((r_1 + r_2) + I\big)(m + IM) = \) \((r_1 + r_2)m + IM = \) \((r_1m + r_2m) + IM = \) \((r_1m + IM) + (r_2m + IM) = \) \(\big((r_1 + I)(m + IM)\big) + \big((r_2 + I)(m + IM)\big)\);
  • \((r + I)\big((m_1 + IM) + (m_2 + IM)\big) = \) \((r + I)\big((m_1 + m_2) + IM\big) = \) \(r(m_1 + m_2) + IM = \) \((rm_1 + rm_2) + IM = \) \((rm_1 + IM) + (rm_2 + IM) = \) \((r + I)(m_1 + IM) + (r + I)(m_2 + IM)\);
  • \((r_1 + I)\big((r_2 + I)(m + IM)\big) = \) \((r_1 + I)(r_2m + IM) = \) \(r_1(r_2m) + IM = \) \((r_1r_2)m + IM = \) \((r_1r_2 + I)(m + IM) = \) \( \big((r_1 + I)(r_2 + I)\big)(m + IM)\).
Portanto, \(\rfrac{M}{IM}\) é um \(\rfrac{R}{I}\)-módulo com o produto dado por \((r + I)(m + IM) = rm + IM\).

Álgebra Não Comutativa

Aula 6 - Revisão de teoria de módulos (parte 3)

Sejam \(p\) um primo, \(\bbZ_{(p)} = \{\frac{a}{p^m} \mid a \in \bbZ, m \ge 0\}\), \(M = \rfrac{\bbZ_{(p)}}{\bbZ}\). Mostre que \(M\) é um \(\bbZ\)-módulo artiniano, mas não é noetheriano.

Primeiramente, observe que \(M = \rfrac{\bbZ_{(p)}}{\bbZ}\) é um \(\bbZ\)-módulo, com as operações usuais, isto é, \(n(\frac{a}{p^n} + \bbZ) = \frac{na}{p^n} + \bbZ\) e \((\frac{a}{p^n} + \bbZ) + (\frac{b}{p^m} + \bbZ) = \frac{ap^m + bp^n}{p^{n+m}} + \bbZ\). Vamos mostrar que \(M\) é artiniano. Para isso, precisamos caracterizar os \(\bbZ\)-submódulos de \(M\). Mostraremos que eles são da forma \(\rfrac{G_n}{\bbZ}\), para algum \(n \in \bbN\), onde \(G_n = \{\frac{a}{p^n} \mid a \in \bbZ\}\) é um \(\bbZ\)-submódulo de \(\bbZ_{(p)}\). Assim, suponha que \(N\) é um \(\bbZ\)-submódulo próprio de \(\rfrac{\bbZ_{(p)}}{\bbZ}\) e seja \(\frac{a}{p^m} + \bbZ \in N\), com \(\mdc(a, p) = 1\). Então, pelo Teorema de Bézout, temos que existem \(x, y \in \bbZ\) tais que \(ax + p^my = 1\). Disso segue que \[ \frac{1}{p^m} + \bbZ = \frac{xa + yp^m}{p^m} + \bbZ = \big(\frac{xa}{p^m} + \bbZ\big) + \big(\frac{yp^m}{p^m} + \bbZ\big) \in N, \] pois \(\frac{a}{p^m} + \bbZ \in N\) e \(\frac{yp^m}{p^m} + \bbZ = y + \bbZ = 0 + \bbZ \in N\). Suponha que existe \(k\) um inteiro não negativo tal que \(\frac{1}{p^k} + \bbZ \in N\) e \(\frac{1}{p^n} + \bbZ \not\in N\), para todo \(n > k\). Seja \(\frac{b}{p^m} + \bbZ \in N\), com \(\mdc(b, p) = 1\). Então \(m \le k\), pela maximalidade de \(k\), e também vale que \(\frac{b}{p^m} = bp^{k-m}\frac{1}{p^k}\). Logo, \(N = \rfrac{G_k}{\bbZ}\). Caso não exista \(k\), um expoente máximo tal que \(\frac{1}{p^k} + \bbZ \in N\), então \(\frac{1}{p^n} + \bbZ \in N\), para todo \(n\). Isso implica que \(N = \rfrac{\bbZ_{(p)}}{\bbZ}\). Portanto, os \(\bbZ\)-submódulos de \(M\) são da forma \(\rfrac{G_n}{\bbZ}\), para algum \(n \in \bbN\). Assim, se \[ \rfrac{G_{k_1}}{\bbZ} \supseteq \rfrac{G_{k_2}}{\bbZ} \supseteq \cdots \supseteq \rfrac{G_{k_i}}{\bbZ} \supseteq \cdots \] é uma cadeia descendente de \(Z\)-submódulos de \(M\), então \(k_i \le k_{i-1}\). Assim, deve existir algum \(r\) tal que \(\rfrac{G_{r}}{\bbZ} = \rfrac{G_{k_n}}{\bbZ}\), para todo \(n \ge r\), pois, caso contrário, encontraríamos uma sequência decrescente indefinida de inteiros não negativos, o que é um absurdo. Portanto, \(M\) é um \(\bbZ\)-módulo artiniano. Por outro lado, \(M\) não é um \(\bbZ\)-módulo noetheriano, porque \[ \rfrac{G_{p}}{\bbZ} \subseteq \rfrac{G_{p^2}}{\bbZ} \subseteq \rfrac{G_{p^3}}{\bbZ} \subseteq \cdots \] é uma cadeia ascendente de \(\bbZ\)-submódulos de \(M\) não estacionária.

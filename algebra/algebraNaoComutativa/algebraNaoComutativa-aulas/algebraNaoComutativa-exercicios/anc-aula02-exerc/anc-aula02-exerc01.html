Álgebra Não Comutativa

Aula 2 - Revisão de teoria de anéis (parte 1)

Seja \(R\) um anel com unidade. Mostre que \(\calZ(M_n(R)) = \{rI_n \mid r \in \calZ(R)\}\).

Sejam \(A \in M_n(R)\) e \(r \in \mathcal{Z}(R)\). Então \(A(rI_n) = rAI_n = (rI_n)A\). Dessa forma, \(rI_n \in \mathcal{Z}(M_n(R))\). Reciprocamente, seja \(C \in \mathcal{Z}(M_n(R))\). Considere as matrizes \(E_{ij}\) onde \[ (E_{ij})_{rs} = \begin{cases} 1, & \text{se } r = i \text{ e } s = j; \\ 0, & \text{caso contrário}, \end{cases} \] para todo \(i, j \in \{1, \dots, n\}\). Como \(C \in \mathcal{Z}(M_n(R))\), então \(E_{ii}C = CE_{ii}\), para todo \(i = 1, \dots, n\). Observe que \[ (E_{ii}C)_{rs} = \sum_{k=1}^{n}(E_{ii})_{rk}(C)_{ks} = \left\{ \begin{array}{cl} (C)_{is}, & \text{se } r = i; \\ 0, & \text{caso contrário}, \end{array} \right. \] ou seja, \(E_{ii}C\) é a matriz cuja linha \(i\) corresponde a linha \(i\) de \(C\) e as demais entradas são zero. Veja também que \[ (CE_{ii})_{rs} = \sum_{k=1}^{n}(C)_{rk}(E_{ii})_{ks} = \left\{ \begin{array}{cl} (C)_{ri}, & \text{se } s = i; \\ 0, & \text{caso contrário}, \end{array} \right. \] isto é, \(CE_{ii}\) é a matriz cuja coluna \(i\) corresponde a coluna \(i\) de \(C\) e as demais entradas são zero. Uma vez que \(E_{ii}C = CE_{ii}\), então exceto \(C_{ii}\) todas as outras entradas tem que ser zero. Como \(i = 1, \dots, n\), então \(C\) tem que ser uma matriz diagonal. Novamente, por \(C\) pertencer ao centro de \(M_n(R)\), temos que \(E_{ij}C = CE_{ij}\). De maneira análoga ao feito anteriormente, temos que

  1. \(E_{ij}C\) é a matriz cuja linha \(i\) corresponde a linha \(j\) da matriz \(C\) e os demais elementos são zero;
  2. \(CE_{ij}\) é a matriz cuja coluna \(j\) corresponde a coluna \(i\) da matriz \(C\) e os demais elementos são zero.
Assim, considerando apenas as possibilidades cujas entradas são não nulas, temos que \((C)_{js} = (E_{ij}C)_{is} = (CE_{ij})_{rj} = (C)_{ri}\). Dessa forma, \(r = i\), \(s = j\) e, portanto, \((C)_{jj} = (C)_{ii}\). Com isso, concluímos que \(C = rI_n\). Finalmente, note que, para todo \(s \in R\), temos que \(rsI_n = (rI_n)(sI_n) = (sI_n)(rI_n) = srI_n\). Portanto, \(r \in \mathcal{Z}(R)\), concluindo o exercício.

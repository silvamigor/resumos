Álgebra Não Comutativa

Aula 17 - Revisão de extensão de corpos

Ao longo dessa aula, \(K\) e \(F\) denotarão corpos.

Dizemos que \(K\) é uma extensão de \(F\), se \(F\) é um subcorpo de \(K\).

Se \(K\) é uma extensão de \(F\), então \(K\) é uma \(F\)-álgebra e \(\dim_F(K) = [K:F]\) é chamado de grau da extensão. A extensão é finita, se \([K:F] < \infty\) e infinita, caso contrário.

Se \(K\) é uma extensão de \(F\) e \(\Gamma \subseteq K\), então o subcorpo (subanel) de \(K\) gerado por \(\Gamma\) é a interseção de todos os subcorpos (subanéis) de \(K\) que contêm \(\Gamma\). O subcorpo de \(K\) gerado por \(\Gamma \cup F\) será denotado por \(F(\Gamma)\). Se \(\Gamma = \{\alpha_1, \dots, \alpha_n\}\), então \(F(\Gamma) = F(\alpha_1, \dots, \alpha_n)\) e se \(\Gamma = \{\alpha\}\), então \(F(\Gamma) = F(\alpha)\). Neste caso, dizemos que \(F(\alpha)\) é um extensão simples de \(F\).

Se \(K\) é uma extensão de \(F\) e \(u \in K\), então dizemos que \(u\) é algébrico sobre \(F\), se existe \(f(x) \in F[x]\) tal que \(f(u) = 0\). Caso contrário, dizemos que \(u\) é transcendente. Se \(u \in K\) é algébrico sobre \(F\), então existe um polinômio mônico irredutível \(g(x) \in F[x]\) tal que \(g(u) = 0\). Chamamos esse polinômio de polinômio minimal de \(u\).

Dizemos que \(K\) é uma extensão algébrica de \(F\), se todo elemento de \(K\) é algébrico sobre \(F\). Dizemos que \(K\) é uma extensão transcendente, se pelo menos um elemento de \(K\) é transcendente sobre \(F\).

Por exemplo, \(i \in \bbC\) é algébrico sobre \(\bbQ\) e, consequentemente, sobre \(\bbR\). Temos que \(\bbC\) é uma extensão algébrica de \(\bbR\) (\(\bbC = \bbR(i)\), \([\bbC:\bbR] = 2\)). Temos que \(\bbR\) é uma extensão transcendente de \(\bbQ\), pois \(\pi\) e \(e\) são transcendentes sobre \(\bbQ\).

Se \(K\) é uma extensão de \(F\) e \(u \in K\) é algébrico sobre \(F\), então \(F(u) = F[u]\) (subanel gerado por \(u\)) e \([F(u): F] = n\), onde \(n\) é o grau do polinômio minimal de \(u\). Além disso, \(\{1, u, u^2, \dots, u^{n-1}\}\) de \(F(u)\) sobre \(F\) e se \(f(x)\) é o polinômio minimal de \(u\), então \(F(u) \cong \rfrac{F[x]}{\langle f(x) \rangle}\).

Se \(K\) é uma extensão finita de \(F\), então \(K\) é uma extensão algébrica de \(F\).

Se \(f(x) \in F[x]\), então dizemos que \(f(x)\) cinde sobre \(F\), se \(f(x)\) pode ser escrito na forma \(u_0(x - u_1)\cdots (x - u_n)\), onde \(u_i \in F\).

Uma extensão \(K\) de \(F\) é um corpo de decomposição de um polinômio \(f(x) \in F[x]\), se \(f(x)\) cinde sobre \(K\) e \(K = F(u_1, \dots, u_n)\), onde \(u_i\) são as raízes de \(f(x)\) em \(K\).

Todo polinômio \(f(x) \in F[x]\) de grau \(n \ge 1\) possui um corpo de decomposição \(K\) sobre \(F\) e \([K:F] \le n!\).

  1. As únicas raízes de \(f(x) = x^2 - 7\) são \(\pm \sqrt{7}\). Então, \(f(x) = (x - \sqrt{7})(x + \sqrt{7})\) e \(\bbQ(\sqrt{7}) = \bbQ(\sqrt{7}, -\sqrt{7})\) é corpo de decomposição de \(f(x)\) sobre \(\bbQ\).
  2. Considere \(g(x) = x^3 - 7\). Seja \(u\) uma raiz real de \(g\). Então \(\bbQ(u) \subseteq \bbR\) não é um corpo de decomposição de \(g(x)\) sobre \(\bbQ\), pois não contém todas as raízes de \(g(x)\).

Seja \(f(x) \in F[x]\) irredutível.

  1. Dizemos que \(f(x)\) é um polinômio separável, se em algum corpo de decomposição de \(f(x)\) sobre \(F\), toda raiz de \(f(x)\) é simples.
  2. Se \(K\) é uma extensão de \(F\) e \(u \in K\) é algébrico sobre \(F\), então dizemos que \(u\) é um elemento separável sobre \(F\), se o polinômio minimal de \(u\) é separável.
  3. Se todo elemento de \(K\) é separável sobre \(F\), então \(K\) é chamada de uma extensão separável de \(F\).

Sejam \(K\) uma extensão de \(F\), \(f(x) \in F[x]\) e \(u \in K\). Temos que \(u\) é uma raiz múltipla de \(f(x)\) se, e somente se, \(f(u) = 0\) e \(f'(u) = 0\).

Um polinômio irredutível \(f(x) \in F[x]\) é separável se, e somente se, \(f'(x) \neq 0\). Em particular, se \(\char(F) = 0\), então todo polinômio irredutível é separável e, portanto, toda extensão algébrica de \(F\) é separável.

Se \(F\) é um corpo, então as seguintes afirmações são equivalentes:

  1. todo polinômio não constante \(f(x) \in F[x]\) possui uma raiz em \(F\),
  2. todo polinômio não constante \(f(x) \in F[x]\) cinde sobre \(F[x]\),
  3. todo polinômio irredutível em \(F[x]\) tem grau \(1\),
  4. se \(K\) é uma extensão algébrica de \(F\), então \(K = F\),
  5. exite um subcorpo \(F'\) de \(F\) tal que \(F\) é uma extensão algébrica de \(F'\) e todo polinômio em \(F'[x]\) cinde sobre \(F\).

Um corpo que satisfaz alguma das condições da Proposição 17.3 é chamado de algebricamente fechado.

Se \(K\) é uma extensão de \(F\), então dizemos que \(K\) é um fecho algébrico de \(F\), se \(K\) é uma extensão algébrica de \(F\) e \(K\) é algebricamente fechado. De maneira equivalente, \(K\) é o fecho algébrico de \(F\), se \(K\) é o corpo de decomposição de todos os polinômios de \(F[x]\).

Todo corpo possui fecho algébrico e quaisquer dois fechos algébricos de \(F\) são isomorfos como espaços vetoriais sobre \(F\).

Um elemento \(a\) de uma \(F\)-álgebra \(A\) é dito algébrico sobre \(F\), se \(a\) é raiz de algum polinômio em \(F[x]\). Dizemos que \(A\) é uma álgebra algébrica sobre \(F\), se todo elemento de \(A\) for algébrico sobre \(F\).

Toda \(F\)-álgebra de dimensão finita sobre \(F\) é uma álgebra algébrica. De fato, se \(\dim_F(A) = n\) e \(a \in A\), então \(\{a, a^2, \dots, a^{n+1}\}\) é linearmente dependente. Logo, existem \(\alpha_i \in F\) tais que \(\alpha_1a + \alpha_2a^2 + \cdots + \alpha_{n+1}a^{n+1} = 0\). Assim, \(f(x) = \sum_{i=1}^{n+1}\alpha_ix^i\) é tal que \(f(a) = 0\).

Se \(A\) é uma álgebra semissimples de dimensão finita sobre um corpo algebricamente fechado \(F\), então \(A \cong M_{n_1}(F) \oplus \cdots \oplus M_{n_k}(F)\).

Se \(K\) é uma extensão finita e separável de \(F\), então exite \(u \in K\) tal que \(K = F(u)\).

  1. Mostre que se \(D\) é uma álgebra de divisão algébrica sobre um corpo algebricamente fechado \(K\), então \(D = K\). (Solução)

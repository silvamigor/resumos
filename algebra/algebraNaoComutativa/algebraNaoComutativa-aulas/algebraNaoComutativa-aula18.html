Álgebra Não Comutativa

Aula 18 - Semissimplicidade do produto tensorial de duas álgebras

Seja \(L\) uma extensão finita de \(F\). Então \(L \otimes_{F} K\) é uma \(K\)-álgebra semissimples, para toda extensão \(K\) de \(F\) se, e somente se, \(L\) é uma extensão separável de \(F\).

Seja \(A\) um \(F\)-álgebra semissimples de dimensão finita. Pelo Teorema de Wedderburn-Artin, temos que \(A \cong M_{n_i}(D_i) \oplus \cdots \oplus M_{n_k}(D_k)\). Logo, \(\calZ(A) \cong \calZ\big(M_{n_i}(D_i) \oplus \cdots \oplus M_{n_k}(D_k)\big) \cong \calZ(D_1) \oplus \cdots \calZ(D_k)\). Nesse caso, dizemos que \(A\) é uma álgebra separável, se cada \(\calZ(D_i)\) é uma extensão separável de \(F\).

Seja \(A\) um \(F\)-álgebra simples de dimensão finita. Pelo Teorema de Wedderburn-Artin, temos que \(A \cong M_n(D)\), \(D\) \(F\)-álgebra de divisão e com isso, \(A\) é uma \(\calZ(A)\)-álgebra central simples. Se \(B\) é uma \(F\)-álgebra, então \(A \otimes_{F} B \cong A \otimes_{\calZ(A)} (\calZ(A) \otimes_{F} B)\).

Se \(A\) é uma \(F\)-álgebra separável, então \(A \otimes_{F} K\) é uma \(K\)-álgebra semissimples, para toda extensão \(K\) de \(F\).

Se \(A\) e \(B\) são \(F\)-álgebras semissimples de dimensão finita tal que \(A\) ou \(B\) é separável, então \(A \otimes_{F} B\) é semissimples.

Seja \(A\) uma álgebra semissimples de dimensão finita sobre um corpo \(F\) de característica zero. Então para toda extensão algébrica \(K\) de \(F\), \(A \otimes_{K}\) é semissimples.

Sejam \(A\) uma \(F\)-álgebra e \(K\) uma extensão separável de \(F\). Então \(J(A \otimes_{F} K) = J(A) \otimes_{F} K\).

  1. Sejam \(A\) uma \(F\)-álgebra e \(K\) uma extensão de \(F\). Mostre que
    1. \(J(A \otimes_{F} K) \cap A \subseteq J(A)\). Se \(K\) é uma extensão algébrica de \(F\) ou \(\dim_F(A) < \infty\), então \(J(A \otimes_{F} K) \cap A = J(A)\);
    2. se \([K:F] = n\), então \((J(A \otimes_{F}K))^{n} \subseteq J(A) \otimes_{F} K\). (Solução)

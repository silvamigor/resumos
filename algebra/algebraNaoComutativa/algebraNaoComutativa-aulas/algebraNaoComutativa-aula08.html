Álgebra Não Comutativa

Aula 8 - Teorema de Wedderburn

Seja \(R\) um anel. São equivalentes:

  1. todo \(R\)-módulo é semissimples;
  2. \(R\) é um anel semissimples;
  3. \(R\) é a soma direta finita de ideais minimais à esquerda.

Seja \(R = L_1 \oplus \dots \oplus L_n\) a decomposição de um anel semissimples como a soma direta de ideais minimais à esquerda. Se \(M\) é um \(R\)-módulo simples, então \(M \cong L_i\), para algum \(i = 1, \dots, n\). Em particular, a menos de isomorfismos, existe apenas uma quantidade finita de \(R\)-módulos simples.

Todo anel semissimples é artiniano e noetheriano.

Dizemos que um \(R\)-módulo \(M\) é homogêneo, se \(M = M_1 \oplus \cdots \oplus M_n\), onde cada \(M_i \le M\) é simples, isomorfo a um \(R\)-módulo fixo \(M'\).

Seja \(R = L_1 \oplus \cdots \oplus L_n\) a decomposição de um anel semissimples. Agrupando os ideais isomorfos que aparecem nessa decomposição, temos que \(R \cong L_1' \oplus \cdots L_m'\), onde cada \(L_j'\) é a soma direta dos ideais isomorfos a \(L_j\) que aparecem na decomposição de \(L\). Note que cada \(L_j'\) é um anel homogêneo. Logo, todo anel semissimples é a soma direta de anéis homogêneos. Em particular, se \(R\) é um anel semissimples e homogêneo, então todos os \(R\)-módulos simples são isomorfos.

  1. Se \(D\) é um anel de divisão, então \(M_n(D)\) é um anel semissimples, pois \(M_n(D) = C_1 \oplus \cdots \oplus C_n\) e cada \(C_i\) é um ideal minimal à esquerda de \(M_n(D)\).
  2. O anel de polinômios \(F[x]\), \(F\) corpo, não é semissimples, pois ele não é artiniano.

Sabemos que se \(V\) é um espaço vetorial de dimensão finita sobre um corpo \(F\), então \(\End_{F}(V) \cong M_n(F)\). Em particular, \(\End_{F}(F) \cong F\). Se \(\{v_1, \dots, v_n\}\) é uma base de \(V\), então as aplicações \(f_i: F \to Fv_i\), \(\alpha \mapsto \alpha v_i\) são isomorfismos, já que \(F\) é um \(F\)-módulo simples e \(V = Fv_1 \oplus \cdots \oplus Fv_n \cong {F \oplus \cdots \oplus F} = F^{(n)}\). Logo,

\(\End_{F}(V) \cong \End_{F}(F^{(n)}) \cong M_n(F) \cong M_n(\End_{F}(F))\)

Seja \(M\) um \(R\)-módulo semissimples de comprimento finito. Então \(\End_{R}(M)\) é isomorfo a uma soma direta finita de anéis de matrizes com coeficientes em anéis de divisão.

O anel oposto de um anel \(R\), denotado por \(R^{op}\), é definido da seguinte maneira: como grupo abeliano \((R^{op}, +) = (R, +)\) e a multiplicação em \(R^{op}\) é dada por \(a \cdot b = ba\), para todos \(a, b \in R\).

  1. \(R\) tem unidade se, e somente se, \(R^{op}\) tem unidade.
  2. \(R\) é um anel de divisão se, e somente se, \(R^{op}\) é de divisão.
  3. \((R^{op})^{op} = R\).
  4. \(R \cong S\) se, e somente se, \(R^{op} \cong S^{op}\).

Dados \(R\) um anel e \(r \in R\), seja \(f_r: R \to R\), \(x \mapsto rx\). Observe que se \(R\) não é comutativo, então \(f_r\) não é um \(R\)-homomorfismo. Considere \(f_r: R \to R\), \(x \mapsto xr\). Então \(f_r\) é um \(R\)-homomorfismo que induz uma aplicação \(g: R \to \End_{R}(R)\), \(r \mapsto f_r\) que não é um homomorfismo de anéis, porque dados \(r, s \in R\), temos que \(g(rs) = f_{rs}: R \to R\), \(x \mapsto r(rs)\) que é igual a \(f_s \circ f_r\) que é diferente de \(g(r)g(s)\). Se considerarmos \(g: R^{op} \to \End_{R}(R)\), \(r \mapsto f_r\), então \(g\) é um homomorfismo de anéis. Temos que

  1. \(g\) é injetiva, pois dados \(r, s \in R\), \(f_r = f_s\) implica que \(r = f_r(1) = f_s(1) = s\);
  2. \(g\) é sobrejetiva, porque dados \(h \in \End_{R}(R)\), então \(h(x) = h(x1) = xh(1) = f_{h(1)}(x)\).
Conclusão: \(R^{op} \cong \End_{R}(R)\).

Se \(D\) é um anel de divisão e \(V\) é um \(D\)-módulo livre de posto finito, então \[ \End_{D}(V) \cong \End_{D}(D^{(n)}) \cong M_n(\End_{D}(D)) \cong M_n(D^{op}) \] é semissimples, pois \(D^{op}\) é um anel de divisão.

Os principais teoremas de álgebra linear são válidos quando se tira a comutatividade do corpo, isto é, se se usa um anel de divisão.

Todo anel semissimples é isomorfismo a uma soma direta finita de anéis de matrizes com coeficientes em anéis de divisão.

Todo anel semissimples é isomorfismo a um soma direta finita de anéis simples.

Se \(R\) é um anel comutativo semissimples, então \(R\) é isomorfismo a uma soma direta finita de corpos.

  1. Mostre que se \(M\) é um \(R\)-módulo e \(n \ge 1\), então \(\End_{R}(M^{(n)}) \cong M_n(\End_{F}(M))\). (Solução)
  2. Mostre que \(M_n(R)^{op} \cong M_n(R^{op})\). Dê um exemplo de anel tal que \(R^{op} \not\cong R\). (Solução)
  3. Mostre a unicidade da decomposição de Wedderburn de um anel semissimples, isto é, mostre que se \[ R \cong \oplus_{i = 1}^{k_1}M_{n_i}(D_i) \cong \oplus_{j = 1}^{k_2}M_{r_j}(D'_j), \] então \(k_1 = k_2\) e, após uma possível reordenação dos índices, \(n_i = r_i\) e \(D_i \cong D'_j\). (Solução)
  4. Mostre que se \(R\) é semissimples, então \(M_n(R)\) também o é. (Solução)
  5. Seja \(R\) um domínio. Mostre que se \(M_n(R)\) é semissimples, então \(R\) é um anel de divisão. (Solução)

Álgebra Não Comutativa

Aula 14 - Conceitos gerais sobre \(R\)-álgebras

Sejam \(R\) um anel comutativo e \(M\) um \(R\)-módulo. Uma aplicação bilinear \(f: M \times M \to M\) é bilinear, se

  1. \(f(m_1 + m_2, m) = f(m_1, m) + f(m_2, m)\),
  2. \(f(m, m_1 + m_2) = f(m, m_1) + f(m, m_2)\),
  3. \(f(rm_1, m_2) = f(m_1, rm_2) = rf(m_1, m_2)\),
para todo \(r \in R\), \(m, m_1, m_2 \in M\). Na Aula 13, definimos uma aplicação bilinear entre bimódulos sobre um anel.

Uma aplicação bilinear \(f\) induz uma operação em \(M\), denotada por \(\ast\):

  1. \((m_1 + m_2)\ast m = m_1 \ast m + m_2 \ast m\)
  2. \(m\ast(m_1 + m_2) = m\ast m_1 + m\ast m_2\)
  3. \((rm_1)\ast m_2 = r(m_1 \ast m_2) = m_1\ast (rm_2)\)

Seja \(R\) um anel comutativo. Um \(R\)-módulo \(A\) munido de uma aplicação bilinear \(f: A \times A \to A\) é chamado de uma \(R\)-álgebra (ou uma álgebra sobre \(R\)).

  1. Em geral, uma álgebra não é um anel, pois não tem a propriedade da associatividade.
  2. Como \(R\) é comutativo, então uma \(R\)-álgebra \(A\) é um \(R\)-bimódulo, com \(ra = ar\), para todo \(r \in R\) e \(a \in A\).

  1. Todo anel é uma \(\bbZ\)-álgebra.
  2. Se \(D\) é um anel de divisão, então \(D\) é uma álgebra sobre \(\calZ(D)\).
  3. Todo anel comutativo é uma álgebra sobre si próprio.

Ao longo do curso, trabalharemos com álgebras sobre corpos. Uma \(F\)-álgebra, com \(F\) sendo um corpo, é um \(F\)-espaço vetorial. A dimensão de uma \(F\)-álgebra é sua dimensão como espaço vetorial sobre \(F\), a qual denotaremos por \(\dim_F(A)\). Uma base de uma \(F\)-álgebra \(A\) é uma base de \(A\) como espaço vetorial sobre \(F\).

  1. \(\bbC\) é uma \(\bbR\)-álgebra com \(\dim_{\bbR}(\bbC) = 2\) e \(\{1, i\}\) é uma base da álgebra \(\bbC\) sobre \(\bbR\).
  2. \(\bbR\) é uma \(\bbQ\)-álgebra, com \(\dim_{\bbQ}(\bbR) = \infty\).
  3. A \(\bbF\)-álgebra de matrizes \(M_n(\bbF)\) tem \(\dim_{\bbF}(M_n(\bbF)) = n^2\) e \(\{e_{ij} \mid 1 \le i, j \le n\}\) é uma base de \(M_n(\bbF)\) sobre \(\bbF\).
  4. \(\bbR^3\) com o produto dado pelo produto vetorial usual é uma \(\bbR\)-álgebra com \(\dim_{\bbR}(\bbR^3) = 3\). Note que esse produto vetorial não dá uma estrutura de anéis para \(\bbR^3\).

Seja \(A\) uma \(\bbF\)-álgebra. Dizemos que \(A\) é

  1. unitária, se existe \(1 \in A\) tal que \(1\ast a = a \ast 1 = a\);
  2. associativa, se \((a\ast b)\ast c = a\ast (b \ast c)\), para todo \(a, b, c \in A\);
  3. de Lie, se \(a\ast a = 0\) e \((a\ast b)\ast c + (b \ast c)\ast a + (c \ast a)\ast b = 0\), para todo \(a, b, c \in A\);
  4. de Jordan, se \(a\ast b = b \ast a\) e \((a^2 \ast b)\ast a = a^2\ast (b \ast a)\), para todo \(a, b \in A\).

  1. Se \(A\) é associativa, então \(A\) é um espaço vetorial com uma estrutura de anel. Nesse caso, vamos denotar \(a\ast b\) por \(ab\).
  2. Se \(A\) é uma álgebra de Lie, denotamos \(a\ast b\) por \([a, b]\). Chamamos, nesse caso, \([a,b]\) de colchete de Lie. Aqui, temos que \([a,a] = 0\) e \([[a,b],c] + [[b,c],a] + [[c,a],b] = 0\). A última igualdade é chamada de identidade de Jacobi.
  3. Se \(A\) é uma álgebra de Jordan, denotamos \(a\ast b\) por \(a\circ b\), o qual é chamado de produto de Jordan. Aqui vale \(a\circ b = b \circ a\) e \((a^2 \circ b)\circ a = a^2\circ (b \circ a)\).

Sejam \(A\) uma álgebra e \(B = \{a_i \mid i \in I\}\) uma base de \(A\). Se \(a_i, a_j \in B\), então \(a_i\ast a_j = \sum_{k \in I}\alpha_{ij}^ka_k\), onde \(a_k \in B\) e \(\alpha_{ij}^k \in F\). As constantes \(\alpha_{ij}^k \in F\) são chamadas constantes estruturais de \(A\).

  1. De agora em diante, a palavra "álgebra" será sinônimo de "álgebra associativa". Logo, uma álgebra é uma \(F\)-espaço vetorial que possui uma estrutura de anel.
  2. Dizemos que uma álgebra \(A\) é comutativa (nil, nilpotente, de divisão, etc), se \(A\) é um anel comutativo (nil, nilpotente, de divisão, etc).

  1. Seja \(A\) uma álgebra. Uma subálgebra \(B\) de \(A\) é um subespaço vetorial de \(A\) que também é um subanel de \(A\).
  2. Um ideal à esquerda (à direita, bilateral) de \(A\) é um subespaço vetorial de \(A\) que também é um ideal à esquerda (à direita, bilateral) de \(A\).

  1. A palavra "ideal" será reservada para ideal bilateral. A noção de ideais maximais e minimais são definidas analogamente.
  2. Seja \(A\) uma álgebra. Então um ideal do anel \(A\) não necessariamente é um ideal da álgebra \(A\).
  3. Se \(A\) é uma álgebra unitária, então todo ideal do anel \(A\) é um ideal da álgebra \(A\). Se \(\alpha \in F\) e \(a \in A\), então \(\alpha a = \alpha (1a) = (\alpha 1) a = a (\alpha 1)\). Logo, se \(I\) é um ideal de um anel \(A\), então \(\alpha I = (\alpha 1)I \subseteq I\).

Se \(I\) é um ideal de \(A\), então \(\rfrac{A}{I}\) é uma álgebra, chamada de álgebra quociente de \(A\) por \(I\).

Sejam \(A\) e \(B\) álgebras e \(f: A \to B\) uma função. Dizemos que \(f\) é um homomorfismo (de álgebras), se

  1. \(f(a_1 + a_2) = f(a_1) + f(a_2)\)
  2. \(f(a_1a_2) = f(a_1)f(a_2)\)
  3. \(f(\alpha a) = \alpha f(a)\)
para todo \(\alpha \in F\), \(a, a_1, a_2 \in A\). Em outros termos, um homomorfismo de álgebras é uma transformação linear que respeita o produto.

As definições de núcleo, imagem, monomorfismo, epimorfismo, isomorfismo são análogos ao caso de anéis e módulos. Além disso, os teoremas do isomorfismo são válidos para álgebras (reveja a Aula 3).

  1. Seja \(A\) uma álgebra de Lie. Mostre que \([a, b] = -[b, a]\), para todo \(a, b \in A\). Mostre que se \(\char(F) \neq 2\), então está propriedade é equivalente a \([a, a] = 0\), para todo \(a \in A\). (Solução)
  2. Seja \(A\) uma álgebra associativa. Mostre que
    1. o produto \([a, b] = ab - ba\), para todo \(a, b \in A\), dá uma estrutura de álgebra de Lie para \(A\).
    2. se \(\char(F) \neq 2\), então o produto \(a \circ b = \frac{1}{2}(ab + ba)\) dá uma estrutura de álgebra de Jordan para \(A\). (Solução)
  3. Mostre que se \(A\) é um \(F\)-álgebra, então \(\End_{F}(A)\) também o é. Além disso, se \(A\) tem dimensão finita, então \(\End_{F}(A) \cong M_n(F)\). (Solução)

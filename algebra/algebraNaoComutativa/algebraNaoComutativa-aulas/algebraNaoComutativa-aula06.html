Álgebra Não Comutativa

Aula 6 - Revisão de teoria de módulos (parte 3)

Dizemos que um conjunto \(S = \{m_i\}_{i \in I}\) de elementos de um \(R\)-módulo \(M\) é linearmente independente (ou \(R\)-livre), se para qualquer combinação linear finita de \(S\) com coeficientes em \(R\), \[ r_1m_{i_1} + \cdots + r_nm_{i_n} = 0, \] temos que \(r_1 = r_2 = \cdots = r_n = 0\).

Dizemos que \(S\) é uma \(R\)-base de \(M\), se \(S\) é \(R\)-livre e gera \(M\). Se um \(R\)-módulo \(M\) possui uma \(R\)-base, então dizemos que \(M\) é \(R\)-livre. Dizemos que \(M\) tem posto finito, se ele possui um \(R\)-base finita.

  1. \(\bbZ_n\) como \(\bbZ\)-módulo não é livre. Para todo \(\overline{r} \in \bbZ_n\), temos que \(n\overline{r} = \overline{0}\) e \(n \neq 0\) em \(\bbZ_n\). Contudo, \(\bbZ_n\) é \(\bbZ_n\)-módulo é livre.
  2. Se \(R\) é um anel com unidade, então \(R \oplus R\) é \(R\)-livre, pois \(\{(1,0), (0,1)\}\) é uma \(R\)-base de \(R \oplus R\).
  3. Se \(I\) é um conjunto de índices e \(R^{(I)} = \oplus_{i \in I}R_i\), onde \(R_i = R\), então \(R^{(I)}\) é livre, com \(R\)-base dada por \(\{e_i \mid i \in I\}\), onde \(e_j = (r_i)_{i \in I}\) tais que \(r_j = 1\) e \(r_i = 0\), para todo \(i \neq j\).

Sejam \(R\) um anel com unidade e \(M\) um \(R\)-módulo. Então \(M\) é livre se, e somente se, \(M\) é a soma direta de uma família de \(R\)-módulos cíclicos, cada um isomorfo a \(R\) como \(R\)-módulos.

Sejam \(R\) um anel com unidade e \(X\) um conjunto e \(M = R^{(X)}\). Na notação utilizada anteriormente, \(\{e_x \mid x \in X\}\) é uma \(R\)-base de \(M\) e \(M \cong \oplus_{x \in X}R_{e_x}\). Identificando \(e_x\) com \(x\), então \(M \cong \oplus_{x \in X}R_x\) e \(X\) é uma \(R\)-base de \(M\). Neste caso, dizemos que \(M\) é o \(R\)-módulo livremente gerado pelo conjunto \(X\).

Uma relação \(\le\) em um conjunto \(X\) é um subconjunto do produto cartesiano \(X \times X\). Dizemos que \((X, \le)\) é um conjunto parcialmente ordenado, se para todo \(a, b, c \in X\), temos que valem as seguintes condições:

  1. reflexividade: \(a \le a\)
  2. antissimetria: se \(a \le b\) e \(b \le a\), então \(a = b\)
  3. transitividade: se \(a \le b\) e \(b \le c\), então \(a \le c\) transitiva
Se \(a \le b\) ou \(b \le a\), então dizemos que \(a\) e \(b\) são comparáveis. Dizemos que um conjunto parcialmente ordenado \((X, \le)\) é totalmente ordenado, se quaisquer dois elementos em \(X\) são comparáveis.

Sejam \((X, \le)\) um conjunto parcialmente ordenado e \(x \in X\). Dizemos que \(x\) é um elemento maximal em \(X\), se, para todo \(y \in X\) que é comparável a \(x\), temos que \(y \le x\). (veja que estamos olhando apenas os elementos comparáveis; podem ter infinitos elementos maximais ou nenhum).

Dizemos que \(Y \subseteq X\) é um cadeia, se \((Y, \le)\) é totalmente ordenado. Uma cota superior em uma cadeia \(Y \subseteq X\) é um elemento \(x \in X\) tal que \(y \le x\), para todo \(y \in Y\).

Se \((X, \le)\) é um conjunto parcialmente ordenado tal que toda cadeia em \(X\) possui uma cota superior em \(X\), então \(X\) contém (pelo menos) um elemento maximal.

Daqui para frente, a menos que se faça menção explícita, vamos trabalhar com anéis com unidade, para podermos fazer demonstrações mais intuitivas (basicamente, os resultados para anéis sem unidade são os mesmos, contudo, nesse caso, as demonstrações são diferentes e mais trabalhosas).

Se \(R\) é um anel, então \(R\) possui ideal maximal à esquerda (direita, bilateral).

Dizemos que um \(R\)-módulo \(M\) é noetheriano, se toda cadeia ascendente de \(R\)-submódulos de \(M\) é estacionária, isto é, se \[ M_1 \subseteq M_2 \subseteq M_3\subseteq \cdots \] então existe \(r \in \bbN\) tal que \(M_i = M_r\), para todo \(i \ge r\).

Dizemos que um \(R\)-módulo \(M\) é artiniano, se toda cadeia descendente de \(R\)-submódulos de \(M\) é estacionária, isto é, se \[ M_1 \supseteq M_2 \supseteq M_3 \supseteq \cdots \] então existe \(r \in \bbN\) tal que \(M_i = M_r\), para todo \(i \ge r\).

Uma anel \(R\) é noetheriano (artiniano) à esquerda, se \(R\) é um \(R\)-módulo noetheriano (artiniano). Idem para à direita.

  1. Todo espaço vetorial de dimensão finita é artiniano e noetheriano.
  2. O anel \(\bbZ\) é um anel noetheriano. De fato, se \(n\bbZ \subseteq m\bbZ\), então \(m\) é um divisor de \(n\). Como a quantidade de divisores de um número é finita, então toda cadeia ascendente de ideais de \(Z\) é estacionária.
  3. O anel \(\bbZ\) não é artiniano, porque, por exemplo,

    \(2\bbZ \supseteq 4\bbZ \supseteq 8\bbZ \supseteq \cdots \supseteq 2^m\bbZ\).

Seja \(M\) um \(R\)-módulo. São equivalentes:

  1. toda família não vazia de \(R\)-submódulos de \(M\) contém um elemento maximal;
  2. todo \(R\)-submódulo de \(M\) é finitamente gerado;
  3. \(M\) é noetheriano.

Seja \(M\) um \(R\)-módulo. São equivalentes:

  1. toda família não vazia de \(R\)-submódulos de \(M\) contém um elemento minimal;
  2. \(M\) é artiniano.

  1. Dê exemplo de:
    1. um conjunto gerador de um \(R\)-módulo \(M\) que não contém uma \(R\)-base de \(M\);
    2. um conjunto \(R\)-livre de um \(R\)-módulo \(M\) que não está contido em nenhuma base de \(M\);
    3. um exemplo de um \(R\)-submódulo de um \(R\)-módulo livre que não é livre. (Solução)
  2. Seja \(R\) um anel com unidade. Mostre que todo \(R\)-módulo é isomorfo ao quociente de algum \(R\)-módulo livre. (Solução)
  3. Sejam \(p\) um primo, \(\bbZ_{(p)} = \{\frac{a}{p^m} \mid a \in \bbZ, m \ge 0\}\), \(M = \rfrac{\bbZ_{(p)}}{\bbZ}\). Mostre que \(M\) é um \(\bbZ\)-módulo artiniano, mas não é noetheriano. (Solução)
  4. Mostre que o anel \[ R = \begin{pmatrix} \bbR & \bbR \\ 0 & \bbQ \end{pmatrix} \] é artiniano à esquerda, noetheriano à esquerda, mas não é artiniano, nem noetheriano à direita. (Solução)
  5. Mostre que todo domínio de integridade artiniano é um corpo. (Solução)

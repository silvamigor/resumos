Álgebra Não Comutativa

Aula 7 - Semissimplicidade

Sejam \(M\) um \(R\)-módulo e \(N \le M\). Então \(M\) é noetheriano (artiniano) se, e somente se, \(N\) e \(\rfrac{M}{N}\) são ambos noetheriano (artiniano).

Seja \(M\) um \(R\)-módulo tal que \(M = M_1 + \cdots + M_k\), onde \(M_i \le M\), \(i = 1, \dots, k\). Então \(M\) é noetheriano (artiniano) se, e somente se, cada \(M_i\) é noetheriano (artiniano).

Se \(R\) é um anel noetheriano (artiniano), então todo \(R\)-módulo finitamente gerado é noetheriano (artiniano).

Seja \(M\) um \(R\)-módulo. Uma série de composição de \(M\) é a menor cadeia finita de \(R\)-submódulos \[ M_0 = \zero \subseteq M_1 \subseteq M_2 \subseteq \cdots \subseteq M_{n-1} \subseteq M_n = M \] tal que cada quociente \(\rfrac{M_{i+1}}{M_i}\) é simples, \(0 \le i \le n - 1\). Um \(R\)-módulo que possui uma série de composição é dito de comprimento finito.

Um \(R\)-módulo \(M\) é semissimples, se todo \(R\)-submódulo de \(M\) é um somando direto, isto é, se \(N \le M\), então existe \(N' \le M\) tal que \(M = N \oplus N'\). Um anel \(R\) é semissimples, se \(R\) é um \(R\)-módulo semissimples.

  1. Todo espaço vetorial de dimensão finita sobre um corpo \(\bbF\) é um \(\bbF\)-módulo semissimples.
  2. \(\bbZ\) como \(\bbZ\)-módulo não é semissimples, pois os únicos somandos diretos de \(\bbZ\) são \(\zero\) e ele próprio.

Qualquer \(R\)-submódulo \(N\) de um \(R\)-módulo \(M\) semissimples é semissimples. Além disso, qualquer \(R\)-módulo \(M\) semissimples possui um \(R\)-submódulo simples.

Se \(M\) é um \(R\)-módulo semissimples e \(N \le M\), então \(\rfrac{M}{N}\) é semissimples.

Seja \(M\) um \(R\)-módulo. São equivalentes:

  1. \(M\) é semissimples;
  2. \(M\) é a soma direta de \(R\)-submódulos simples;
  3. \(M\) é a soma de \(R\)-submódulos simples.

A soma direta de \(R\)-módulos semissimples é semissimples.

Seja \(M = \oplus_{i \in I}M_i\) a decomposição em simples de um \(R\)-módulo semissimples. Se \(N \le M\), então existe \(J \subseteq I\) tal que \(N = \oplus_{j \in J}M_j\).

  1. Dê um exemplo de um \(R\)-módulo finitamente gerado que possui um \(R\)-submódulo que não é finitamente gerado. (Solução)
  2. Mostre que um \(R\)-módulo \(M\) tem comprimento finito se, e somente se, é artiniano e noetheriano. (Solução)
  3. Mostre que se \(D\) é um anel de divisão, então \(M_n(D)\) é um anel artiniano e noetheriano, para todo \(n \ge 1\). (Solução)
  4. Dê um exemplo de um \(R\)-módulo \(M\) que possui um \(R\)-submódulo \(N\) tal que \(N\) e \(\rfrac{M}{N}\) são semissimples, mas \(M\) não o é. (Solução)
  5. Mostre que o \(\bbZ\)-módulo \(\bbQ\) não é semissimples e não possui nenhum quociente simples. (Solução)
  6. Seja \(M\) um \(R\)-módulo semissimples. Mostre que são equivalentes:
    1. \(M\) é finitamente gerado;
    2. \(M\) é a soma direta finita de \(R\)-módulos simples;
    3. \(M\) tem comprimento finito;
    4. \(M\) é noetheriano e artiniano. (Solução)
  7. Mostre que um \(R\)-módulo é semissimples se, e somente se, todo \(R\)-submódulo cíclico é semissimples. (Solução)

Álgebra Não Comutativa

Aula 13 - Produto tensorial

  1. Vamos denotar um \(R\)-módulo à esquerda \(M\), por \({}_RM\). Se \(M\) for um \(R\)-módulo à direita, vamos denotá-lo por \(M_R\).
  2. Se \(R\) é comutativo e \(M\) é um \(R\)-módulo à esquerda, então \(M\) é um \(R\)-módulo à direita se definirmos \(mr = rm\), para todo \(r \in R\) e para todo \(m \in M\) (veja o Exercício 4.1).
  3. Se \(M\) e \(N\) são \(R\)-módulos, então \(M \times N\) é um \(R\)-módulo, se definirmos
\(r(m,n) = (rm,rn)\)
\((m_1, n_1) + (m_2, n_2) = (m_1+m_2,n_1+n_2)\)

Sejam \(M_R\), \({}_RM\) \(R\)-módulos e \(L\) um \(\bbZ\)-módulo livremente gerado pelo \(M \times N\). Seja \(W\) o \(\bbZ\)-submódulo de \(L\) gerado pelos seguintes elementos:

  1. \((m_1+m_2,n)-(m_1,n)-(m_2,n)\)
  2. \((m,n_1+n_2)-(m,n_1)-(m,n_2)\)
  3. \((mr,n)-(m,rn)\)
para todo \(m, m_1, m_2 \in M\), \(n, n_1, n_2 \in N\) e \(r \in R\). O \(\bbZ\)-módulo quociente \(\rfrac{L}{W}\) é chamado de produto tensorial entre os módulos \(M\) e \(N\) e é denotado por \(M \otimes_R N\). A classe \(\overline{(m,n)} \in T\) é denotada por \(m \otimes n\) e a classe \(\overline{(0,0)}\) por \(0\).

  1. \((m_1+m_2)\otimes n = m_1\otimes n + m_2\otimes n\).
  2. \(m \otimes (n_1 + n_2) = m \otimes n_1 + m \otimes n_2\).
  3. \((mr) \otimes n = m \otimes rn\).

Para todo \(m \ge 2\), \(\bbQ \otimes_{\bbZ} \bbZ_m = \zero\). Esse exemplo, mostra que podemos ter \(m\otimes n = m' \otimes n'\), mas \(m \neq m'\) e \(n \neq n'\).

Sejam \(M_R\), \({}_RN\) \(R\)-módulos e \(P\) um \(\bbZ\)-módulo. Dizemos que uma aplicação \(f: M \times N \to P\) é uma aplicação balanceada, se satisfaz

  1. \(f(m_1 + m_2, n) = f(m_1, n) + f(m_2,n)\);
  2. \(f(m, n_1 + n_2) = f(m, n_1) + f(m,n_2)\);
  3. \(f(rm,n) = f(m,rn)\).

Temos que a aplicação \[ \begin{array}{rcl} b: M \times N &\to& M \otimes_R N \\ (m,n) &\mapsto& m \otimes n \end{array} \] é uma aplicação balanceada, chamada de aplicação balanceada canônica.

Sejam \(M_R\), \({}_RN\) \(R\)-módulos e \(P\) um \(\bbZ\)-módulo. Seja \(f: M \times N \to P\) uma aplicação balanceada. Então existe um único \(\bbZ\)-homomorfismo \(\overline{f}: M \otimes_R N \to P\) tal que \(\overline{f}\circ b = f\). \[ \xymatrix{ M \times N \ar[r]^{b} \ar[dr]_{f} \ar@{}[dr]^{\circlearrowleft} & M \otimes_R N \ar[d]^{\overline{f}} \\ & P } \] Além disso, o produto tensorial é unicamente determinado, a menos de isomorfismos, por esta propriedade.

Sejam \(R\) e \(S\) anéis e \((M, +)\) um grupo abeliano. Dizemos que \(M\) é um \((R,S)\)-bimódulo, se \(M\) é simultaneamente um \(R\)-módulo à esquerda, um \(S\)-módulo à direita e satisfaz \((rm)s = r(ms)\), para todo \(r \in R\), \(s \in S\) e \(m \in M\). Quando \(R=S\), simplesmente dizemos que \(M\) é um \(R\)-bimódulo. Notação: \({}_RM_S\).

Sejam \(R\) e \(S\) anéis e \({}_SM_R\), \({}_RN_S\) bimódulos.

  1. \(M \otimes_R N\) é um \(S\)-módulo tal que \(s(m \otimes n) = (sm) \otimes n\)
  2. Se \(f: M \to M'\) é um homomorfismo de \((S,R)\)-bimódulos e \(g: N \to N'\) é um \(R\)-homomorfismo, então \(f \otimes g: M \otimes_R N \to M' \otimes_R N'\), \(m \otimes n \mapsto f(m) \otimes g(n)\) é um \(S\)-homomorfismo.

  1. Se \(\{RM_i \mid i \in I\}\) uma família de \(R\)-módulos e \(N_R\) é um \(R\)-módulo, então \(N \otimes_R (\oplus_{i \in I}M_i) \cong \oplus_{i\in I}(N \otimes_R M_i)\).
  2. Se \(R\) é um anel com unidade e \(M\) é um \(R\)-módulo, então \(R \otimes_R M \cong M\).
  3. Se \(M\) e \(N\) são \(R\)-bimódulos, então \(M \otimes_R N \cong N \otimes_R M\).
  4. Se \(M_R\), \(RL\), \(RN_S\), então \(M \otimes_R (N \otimes_R L) \cong (M \otimes_R N) \otimes_R L\).

Se \(R\) é comutativo, então \(r(m \otimes n) = \)\(rm \otimes n = \)\(mr \otimes n = \)\(m \otimes rn = \)\((m \otimes n)r\), ou seja, \(M \otimes_R N\) é um \(R\)-bimódulo.

Sejam \(R\) um anel comutativo e \(M, N, P\) \(R\)-bimódulos. Uma aplicação \(f:M \times N \to P\) é bilinear, se

  1. \(f(m_1+m_2,n) = f(m_1,n)+f(m_2,n)\)
  2. \(f(m, n_1+n_2) = f(m,n_1)+f(m,n_2)\)
  3. \(f(rm,n) = f(m,rn) = rf(m,n)\)

Toda aplicação bilinear se fatora pela balanceada.

Se \(R\) é um anel com unidade e \(M_R\), \({}_RN\) são \(R\)-módulos livres com \(R\)-bases \(X\) e \(Y\), respectivamente, então \(M \otimes_R N\) é livre com base \(Z = \{x \otimes y \mid x \in X, y \in Y\}\) de cardinalidade \(|X||Y|\).

Sejam \(R\) um anel com unidade e \(S\) um subanel de \(R\) que contém a unidade \(1_R\). Se \(M\) é um \(S\)-módulo livre com \(S\)-base \(X\), então \(R \otimes_S M\) é um \(R\)-módulo livre, com base \(\{1_R \otimes x, x \in X\}\) e cardinalidade \(|X|\).

Sejam \(F\) um corpo, \(K\) uma extensão de \(F\) e \(V\) um \(F\)-espaço vetorial. Então \(V^K = V \otimes_F K\) é um \(K\)-espaço vetorial com base \(B^K = \{v \otimes 1 \mid v \in B\}\), onde \(B\) é uma \(F\)-base de \(V\) e \(\dim_F(V) = \dim_K(V^K)\).

Álgebra Não Comutativa

Aula 4 - Revisão de teoria de módulos (parte 1)

Sejam \(R\) um anel e \(M\) um grupo abeliano. Um \(R\)-módulo é o grupo \(M\), onde para cada \(r \in R\) e \(m \in M\) está associado o elemento \(rm \in M\) tal que

  1. \((r_1 + r_2)m = r_1m + r_2m\);
  2. \(r(m_1 + m_2) = rm_1 + rm_2\);
  3. \(r_1(r_2m) = (r_1r_2)m\)
para todo \(r_1, r_2 \in R\) e para todo \(m_1, m_2 \in M\). Além disso, se \(R\) tem unidade, então \(1m = m\), para todo \(m \in M\). Para alguns autores, a condição anterior não é necessária mesmo \(R\) tendo unidade. Para eles, nesse caso, o \(R\)-módulo é chamado de \(R\)-módulo unitário.

  1. \(R\)-módulo é sinônimo de \(R\)-módulo à esquerda.
  2. Tudo que for demonstrado para \(R\)-módulos à esquerda pode ser demonstrado para \(R\)-módulos à direita.

  1. \(r0_M = 0_M\), para todo \(r \in R\)
  2. \(0_Rm = 0_M\), para todo \(m \in M\)
  3. \((-r)m = -(rm) = r(-m)\), para todo \(r \in R\) e para todo \(m \in M\)
  4. \(a(rm) = r(am)\), para todo \(a \in \bbZ\), para todo \(r \in R\) e para todo \(m \in M\)

  1. Anéis e grupos abelianos são \(\bbZ\)-módulos.
  2. Quando \(R\) é um corpo, então \(R\)-módulos são os espaços vetoriais sobre \(R\).
  3. Sejam \(R\) um anel e \(S\) um subanel de \(R\). Então \(R\) é um \(S\)-módulo, mas, em geral, \(S\) não é um \(R\)-módulo. No entanto, se \(S\) for um ideal à esquerda de \(R\), então \(S\) não é um \(R\)-módulo. Em particular, \(R\) é um \(R\)-módulo.
  4. Se \(I \unlhd_{\ell} R\), então \(\rfrac{R}{I}\) é um grupo abeliano e possui estrutura de \(R\)-módulo, se definirmos \(r\overline{r_1} = \overline{rr_1}\), para todo \(r, r_1 \in R\).
  5. Se \(f: R \to S\) é um homomorfismo de anéis, então todo \(S\)-módulo \(M\) pode ser visto como um \(R\)-módulo via \(rm := f(r)m\), para todo \(r \in R\) e para todo \(m \in M\).

Sejam \(M\) um \(R\)-módulo e \(N\) um subconjunto não vazio de \(M\). Dizemos que \(N\) é um \(R\)-submódulo de \(M\), denotado por \(N \le M\), se para todo \(n_1, n_2 \in N\) e para todo \(r \in R\), temos que

  1. \(n_1 + n_2 \in N\)
  2. \(rn_1 \in N\)

  1. Os \(R\)-submódulos de um anel \(R\) são os seus ideais à esquerda.
  2. Se \(R\) é um corpo e \(V\) é um \(R\)-espaço vetorial, então os \(R\)-submódulos de \(V\) são seus subespaços vetoriais.
  3. Sejam \(M\) um \(R\)-módulo e \(N \le M\). Como no caso de anéis, o quociente \(\rfrac{M}{N}\) tem estrutura de \(R\)-módulo definindo \(r\overline{m} = \overline{rm}\), para todo \(r \in R\) e para todo \(m \in M\).
  4. Sejam \(M\) um \(R\)-módulo, \(I \unlhd_{\ell} R\) e \(N\) um subconjunto não vazio de \(M\). Então o conjunto \[ IN = \{\sum_{i=1}^{k}r_in_i \mid r_i \in I, n_i \in N, k \ge 1\} \] é um \(R\)-submódulo de \(M\). Em particular, se \(N = \{n\}\), então \(IN = In = \{rn \mid r \in I\}\) é um \(R\)-submódulo de \(M\).

Se \(\{N_i \mid i \in I\}\) é uma família de \(R\)-submódulos de um \(R\)-módulo \(M\), então \(\cap_{i\in I}N_i \le M\).

Se \(X\) é um subconjunto não vazio de um \(R\)-módulo \(M\), então a interseção de todos os \(R\)-submódulos de \(M\) que contém \(X\) é um submódulo de \(M\), chamado de \(R\)-submódulo gerado por \(X\) e denotado por \(\langle X \rangle\). Se \(X = \{m\}\), então \(\langle X \rangle = \langle m \rangle\). Nesse caso, dizemos que \(X\) é cíclico. Como no caso de anéis:

  1. \(\langle m \rangle = \{rm + nm \mid r \in R, n \in \bbZ\}\). Se \(R\) tem unidade, então \(\langle m \rangle = Rm = \{rm \mid r \in R\}\)
  2. \(\langle X \rangle = \{\sum_{i=1}^{s}r_im_i + \sum_{j=1}^{t}n_jm_j \mid m_i,m_j \in X, r_i \in R, n_j \in \bbZ, s,t \ge 0\}\). Se \(R\) tem unidade, então \(\langle X \rangle = RX\). Se \(M = \langle X \rangle\) para algum subconjunto não vazio \(X\) de \(M\), então dizemos que \(X\) gera \(M\). Se \(X\) é finito, então dizemos que \(M\) é um \(R\)-módulo finitamente gerado.

Por abuso de notação, usaremos \(RX\) para denotar o \(R\)-módulo gerado por \(X\).

Seja \(f: M \to N\) uma aplicação entre dois \(R\)-módulos. Dizemos que \(f\) é um homomorfismo de \(R\)-módulos (ou \(R\)-homomorfismo), se para todos \(r \in R\), \(m,n \in M\), temos que

  1. \(f(m+n) = f(m) + f(n)\)
  2. \(f(rm) = rf(m)\)

  1. Se \(R\) é um corpo, então \(R\)-homomorfismos são aplicações lineares.
  2. As noções de monomorfismo, epimorfismo, isomorfismo (\(M \cong N\)), endomorfismo e automorfismo (veja a Definição 3.2) são definidas analogamente como no caso de anéis, assim como núcleo e imagem (veja a Definição 3.3). Além disso, \(\ker(f) \le M\) e \(\im(f) \le N\).
  3. Todos os "teoremas do isomorfismo de anéis" são válidos para \(R\)-módulos (veja o Teorema 3.1).
  4. O conjunto \(\End_R(M) = \{f: M \to M \mid f\) é um endomorfismo\(\}\) é um anel com unidade e possui estrutura de \(R\)-módulo se definirmos \((rf)(m) = rf(m)\), para todo \(r \in R\) e para todo \(m \in M\)
  5. Se \(M, N\) são \(R\)-módulos, então o conjunto \(\Hom_R(M,N) = \{f: M \to N \mid f\) é um \(R\)-homomorfismo\(\}\) é um \(R\)-módulo. Se \(M = N\), então \(\Hom_R(M,M) = \End_R(M)\)

Dizemos que um \(R\)-módulo \(M\) é um \(R\)-módulo simples, se \(RM \neq \zero\) e os únicos \(R\)-submódulos de \(M\) são \(\zero\) e \(M\).

  1. Cuidado! Se \(D\) é um anel de divisão, então \(M_n(D)\) é um anel simples. Contudo, \(M_n(D)\) visto como \(M_n(D)\)-módulo, não é simples, pois os conjuntos \(C_k\) são \(M_n(D)\)-submódulos de \(M_n(D)\)-módulo. Assim, podemos ter um anel simples, mas não é um módulo simples sobre ele mesmo.
  2. Em anéis com unidade, não precisa da condição \(RM \neq \zero\).

  1. Sejam \(R\) um anel comutativo e \(M\) um \(R\)-módulo (enfatizando, à esquerda). Mostre que o produto dado por \(mr := rm\), para todo \(r \in R\) e para todo \(m \in M\), dá uma estrutura de \(R\)-módulo à direita a \(M\) (e vice-versa). (Solução)
  2. Sejam \(I \unlhd R\) (bilateral) e \(M\) um \(R\)-módulo. Mostre que \(\rfrac{M}{IM}\) tem estrutura de \(\rfrac{R}{I}\)-módulo com o produto dado por \((r + I)(m + IM) = rm + IM\). (Solução)
  3. Sejam \(V\) um \(R\)-espaço vetorial e \(T \in \End(V)\). Se \(f(x) \in \sum_{i=0}^{n}a_ix^i \in R[x]\), defina \(f(T) = \sum_{i=0}^{n}a_iT^i \in \End(V)\). Considere \(V\) como um \(R[x]\)-módulo com a estrutura definida por \(f(x)v = f(T)(v)\), para todo \(v \in V\) e \(f(x) \in R[x]\). Mostre que os \(R[x]\)-submódulos de \(V\) são exatamente os subespaços de \(V\) que são \(T\)-invariantes (\(T(W) \subseteq W\)). (Solução)

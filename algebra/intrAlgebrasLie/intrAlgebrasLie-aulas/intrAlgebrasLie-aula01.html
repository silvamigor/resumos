Introdução às Álgebras de Lie

Aula 1 - Conceitos básicas

Definição 1.1. Uma álgebra é um $\mathbb{F}$-espaço vetorial $V$ munido de uma operação bilinear, chamada de produto e denotada por $\cdot$, isto é, uma operação $\cdot: V \times V \to V$ tal que

  • $a \cdot (\alpha b + \beta c) = \alpha (a \cdot b)$ $ + \beta (a \cdot c)$,
  • $(\alpha a + \beta b) \cdot c = \alpha (a \cdot c)$ $ + \beta (b \cdot c)$,
para todo $\alpha$, $\beta \in \mathbb{F}$ e para todo $a$, $b$, $c \in V$. Para fins de simplificação, sempre que for possível, denotamos o produto $a \cdot b$, apenas pela justaposição dos elementos, ou seja, por $ab$. Uma álgebra é chamada de:
  • álgebra associativa, se $(ab)c = a(bc)$, para todo $a$, $b$, $c \in V$;
  • álgebra com identidade, se existe $1 \in V$ tal que $1a = a1 = a$, para todo $a \in V$;
  • álgebra comutativa, se $ab - ba = 0$, para todo $a$, $b \in V$.

Definição 1.2. Uma álgebra de Lie é uma álgebra $\mathfrak{g}$, onde o produto, denotado por $[\cdot, \cdot]$ e dito colchete de Lie, satisfaz as condições:

  • antissimetria: $[a, a] = 0$, para todo $a \in \mathfrak{g}$
  • identidade de Jacobi: $[a,[b,c]] + [b, [c, a]] + [c, [a, b]]$ $ = 0$, para todo $a$, $b$, $c \in \mathfrak{g}$.

Observação 1.1. Se $\mathfrak{g}$ é uma álgebra de Lie, então $[a, b] + [b, a] = 0$, para todo $a$, $b \in \mathfrak{g}$.

Exemplo 1.1. São álgebras de Lie os seguintes exemplos:

  • Um $\mathbb{F}$-espaço vetorial $\mathfrak{g}$ com o colchete de Lie definido como $[a, b] = 0$, para todo $a$, $b \in \mathfrak{g}$. Tal álgebra de Lie é chamada álgebra de Lie abeliana.
  • O $\mathbb{R}$-espaço vetorial $\mathbb{R}^3$ com o colchete de Lie definido como $[a, b] = a \times b$, para todo $a$, $b \in \mathfrak{g}$, onde $a \times b$ é o produto vetorial entre $a$ e $b$.

Definição 1.3. Um morfismo entre álgebras de Lie $\mathfrak{g}_1$ e $\mathfrak{g}_2$ é uma transformação linear $f: \mathfrak{g}_1 \to \mathfrak{g}_2$ tal que $f([a, b]) = [f(a), f(b)]$, para todo $a$, $b \in \mathfrak{g}_1$. Um isomorfismo entre álgebras de Lie é um morfismo invertível entre tais álgebra de Lie.

Observação 1.2. Sejam $S_1$, $S_2$ subconjuntos de uma álgebra de Lie $\mathfrak{g}$. Denotado o conjunto $\{[s_1, s_2] \mid s_1 \in S_1, s_2 \in S_2\}$ $\subseteq \mathfrak{g}$ por $[S_1, S_2]$.

Definição 1.4. Sejam $\mathfrak{g}$ uma álgebra de Lie e $\mathfrak{m}$ um subespaço de $\mathfrak{g}$. Dizemos que $\mathfrak{m}$ é

  • uma subálgebra, se $[\mathfrak{m}, \mathfrak{m}] \subseteq \mathfrak{m}$;
  • um ideal, se $[\mathfrak{m}, \mathfrak{g}] \subseteq \mathfrak{m}$.

Observação 1.3. $[\mathfrak{g}, \mathfrak{g}]$ é um ideal da álgebra de Lie $\mathfrak{g}$.

Observação 1.4. Se $\mathfrak{m}$ é um ideal de $\mathfrak{g}$, então ${}^{\mathfrak{g}}/{}_{\mathfrak{m}}$ é uma álgebra de Lie com o colchete de Lie definido como $[a + \mathfrak{m}, b + \mathfrak{m}] = [a, b] + \mathfrak{m}$.

Proposição 1.1. Se $f: \mathfrak{g}_1 \to \mathfrak{g}_2$ é um morfismo de álgebras de Lie, então

  1. $\text{ker}(f)$ é um ideal de $\mathfrak{g}$;
  2. $\text{im}(f)$ é subálgebra de $\mathfrak{g}_2$;
  3. $\text{im}(f) \cong {}^{\mathfrak{g}_1}/{}_{\text{ker}(f)}$.

Proposição 1.2. Seja $(A, \cdot)$ uma álgebra associativa. Então, definindo $\mathfrak{g} := A$ e fazendo $[a, b] := a \cdot b - b \cdot a$, temos que $(\mathfrak{g}, [\cdot, \cdot])$ é uma álgebra de Lie.

Exemplo 1.2. Seja $V$ um $\mathbb{F}$-espaço vetorial. A álgebra de Lie obtida a partir da álgebra associativa $(\text{End}(V), \circ)$, conforme a Proposição 1.2. é denotada por $(\mathfrak{gl}(V), [\cdot,\cdot])$. No caso em que $V = \mathbb{F}^n$, então a álgebra de Lie é denotada por $(\mathfrak{gl}_n(\mathbb{F})$.

Observação 1.5.

  1. Uma álgebra de Lie de dimensão $1$ é, a menos de isomorfismo, a álgebra de Lie abeliana.
  2. Um morfismo $f$ de uma álgebra de Lie $\mathfrak{g}$ para a álgebra de Lie abeliana é um elemento de $\mathfrak{g}^{\ast} := \text{Hom}_{\mathbb{F}}(\mathfrak{g}, \mathbb{F})$ tal que $f([a, b]) = [f(a), f(b)] = 0$, para todo $a$, $b \in \mathfrak{g}$. Logo, em particular, $f|_{[\mathfrak{g},\mathfrak{g}]} = 0$. Um morfismo com essa propriedade, isto é, $f \in \mathfrak{g}^{\ast}$ tal que $f|_{[\mathfrak{g},\mathfrak{g}]} = 0$ é chamado caracter de $\mathfrak{g}$.

Lema 1.1. A transformação linear traço $\text{tr}: \mathfrak{gl}(V) \to \mathbb{F}$ é um caracter.

Exemplo 1.3. O kernel de $\text{tr}$, denotado por $\mathfrak{sl}_n := \{A \in \text{Mat}_{n \times n}(\mathbb{F}) \mid \text{tr}(A) = 0\}$, é uma subálgebra de $\mathfrak{gl}(V)$.

Observação 1.6. Seja $B: V \otimes V \to \mathbb{F}$ uma forma bilinear. Denotamos $O_{B,V} := \{A \in \mathfrak{gl}(V) \mid B(Av, w)$ $ + B(v, Aw) = 0,$ para todo $v, w \in V\}$.

Lema 1.2. $O_{B,V}$ é álgebra de Lie.

Observação 1.7. Seja $B: V \otimes V \to \mathbb{F}$ uma forma bilinear. Fixando uma base para $V$, podemos escrever $B$ segundo a matriz $[B]$. Daí $B(v, w) = v^t[B]w$. Nesse caso, supondo que $V \cong \mathbb{F}^n$, então definimos $O_{B,n} := \{A \in \text{Mat}_{n \times n}(\mathbb{F}) \mid A^tB$ $ + BA = 0\}$.

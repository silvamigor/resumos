Introdução às Álgebras de Lie

Aula 17 - Classificação das álgebras de Lie semissimples

Seja \(Q\) um reticulado de \(V\). Então existe \(\epsilon: Q \times Q \to \{\pm 1\}\) tal que

  1. \(\epsilon(\alpha,\beta + \gamma) = \epsilon(\alpha,\beta)\epsilon(\alpha,\gamma)\),
  2. \(\epsilon(\alpha + \beta, \gamma) = \epsilon(\alpha,\gamma)\epsilon(\beta,\gamma)\),
  3. \(\epsilon(\alpha,\alpha) = (-1)^{\frac{(\alpha,\alpha)}{2}}\).

Demonstração

Seja \(\Pi = \{\alpha_{1},\dots,\alpha_{r}\}\). Lembre-se de que \(Q = \bbZ\Pi\). Assim, para cada \(\alpha_{i}, \alpha_{j} \in Pi\), com \(i \neq j\), defina

  1. \(\epsilon(\alpha_{i}, \alpha_{j}) = \epsilon(\alpha_{j}, \alpha_{i})\), se \((\alpha_{i}, \alpha_{j})\) é par,
  2. \(\epsilon(\alpha_{i}, \alpha_{j}) = -\epsilon(\alpha_{j}, \alpha_{i})\), se \((\alpha_{i}, \alpha_{j})\) é ímpar.
Se \(i = j\), defina \(\(\epsilon(\alpha_{i}, \alpha_{i}) = (-1)^{(\alpha_{i},\alpha_{i})}\). Defina também \(\epsilon(a\alpha_{i},b\alpha_{j}) = \epsilon(\alpha_{i},\alpha_{j})\), para todo \(a, b \in \{\pm 1\}\). Estendendo \(\epsilon\) bimultiplicativamente em \(Q\), temos que \(\epsilon\) satisfaz as condições do enunciado.


Considere a função \(\epsilon\) da Proposição 17.1. Então,

\[ \epsilon(\alpha+\beta,\alpha+\beta) = \epsilon(\alpha,\alpha)\epsilon(\alpha,\beta)\epsilon(\beta,\alpha)\epsilon(\beta,\beta) \]

Logo, \(\epsilon(\alpha,\beta) = (-1)^{(\alpha,\beta)}\epsilon(\beta\alpha)\), para todo \(\alpha, \beta \in Q\).

Se \(L\) é simples de dimensão finita, então \(L\) é isomorfa a uma das seguintes álgebras:

  1. \(\fraksl_{n}\), com \(n \ge 2\),
  2. \(\frakso_{n}\), com \(n \ge 7\),
  3. \(\fraksp_{2n}\), com \(n \ge 2\),
  4. \(E_{6}\),
  5. \(E_{7}\),
  6. \(E_{8}\),
  7. \(F_{4}\),
  8. \(G_{2}\).

Demonstração

Já sabemos que se \(L\) é simples e de dimensão finita, então \(L \cong L(A)\), para alguma matriz de Cartan. Já sabemos também que se \(A\) é do tipo \(A\), \(B\), \(C\) ou \(D\), então \(L(A)\) é de dimensão finita. Logo, falta ver que \(L(A)\) tem dimensão finita se \(A\) for do tipo \(E\), \(F\) ou \(G\). Vamos começar pelo tipo \(E\) (nesse caso, \(\Delta = \{\alpha \in Q = \bbZ\Pi \mid (\alpha,\alpha) = 2\}\)). Seja \(H = \bbK \otimes_{\bbZ} Q\) e, para cada \(\alpha \in \Delta\), considere um gerador \(e_{\alpha}\). Defina \(L = H \oplus (\oplus_{\alpha \in \Delta} \bbK\{e_{\alpha}\})\) tal que

  1. \([h,h'] = 0\), para todo \(h, h' \in H\),
  2. \([h,e_{\alpha}] = -[e_{\alpha},h] = \alpha(h)e_{\alpha}\),
  3. \([e_{\alpha},e_{-\alpha}] = h_{\alpha}'\) (\(h_{\alpha}'\) vem do isomorfismo induzido do produto interno de \(H\) - veja o Observação 8.5),
  4. \([e_{\alpha},e_{\beta}] = \epsilon(\alpha,\beta)e_{\alpha + \beta}\), se \(\alpha + \beta \in \Delta\),
  5. \([e_{\alpha},e_{\beta}] = 0\), se \(\alpha + \beta \not\in \Delta \cup \zero\).
Temos que mostrar que \(L\) é álgebra de Lie. A simplicidade segue do Teorema 10.2, pelo fato de \(\Delta\) ser indecomponível. Para mostrar que \(L\) é uma álgebra de Lie, temos que ver que \([\cdot, \cdot]\) é antissimétrica e satisfaz a identidade de Jacobi. Isso segue da Proposição 17.1 e da definição de \([\cdot,\cdot]\) (para mais detalhes, ver Kac, Lecture 20). Feito isso, produzimos \(3\) álgebras de Lie simples de dimensão finita \(L_{E_{6}}\), \(L_{E_{7}}\) e \(L_{E_{8}}\). Falta ver somente que \(L(A_{F_{4}})\) e \(L(A_{G_{2}})\) têm dimensão finita. Isso também pode ser visto nas notas do Kac, Lecture 20.


As restrições no Teorema 17.1, se dão devido aos isomorfismos:

  1. \(\fraksl_{2} \cong \frakso_{3} \cong \fraksp_{2}\),
  2. \(\frakso_{4} \cong \fraksl_{2} \oplus \fraksl_{2}\),
  3. \(\frakso_{5} \cong \fraksp_{4}\),
  4. \(\frakso_{6} \cong \fraksl_{4}\).
